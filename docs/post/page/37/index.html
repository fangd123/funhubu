所以，我在中科院了。

十月 18, 2018 发布在 未分类
所以，我在中科院了。所以，我在中科院了。 原创：周思宁+李安 玩转湖大 定制故事001真实性：99.9% 故事人：2013级 资环学院 周思宁 来自主人公的话：感谢黄老师的教导，感谢大学给予的挫折让我成长让我强大。感谢中科院的机会，最重要的是，谢谢一路上有你❤ 所有关于「学霸」故事的开头，似乎都是他们并不认为自己是「学霸」。非常不幸，我也算是其中之一。不谦虚地说，今年四月我已经在思考准备保研何处的事情了。大一大二都是班上第一，大三上的绩点是94.13，湖大本校保研的似乎够了。然而继续坐在这口「井」里，像已经度过的、没有多少讲座、没有多少专业交流的平淡三年一样的日子，我已经，受够了。在网上反复地查了又查，还可以试试浙大和南大。 有条评论突然跳进我的眼帘「中科院生态中心是排专业第一的吧。」 好像有什么东西怦然炸开了。炸在我心里。 大二下学期，我去了武汉植物园做实习生。所以某种意义上来说，从那时起，我就加入中科院了。然而在我们院的考研史上，只有一位04级的学长真正试过。 生态中心只回复了一封邮件：不招收双非院校学生。 说实在话，我们院考研的，不乏清华浙大之类的好学校。我那时就想，能考上的我都不报了，要报就报最好的。于是毅然，非生态中心不去。 4月 >>「稻草」何处来？ 跟我的植物园导师H老师说了我的考研念头，她让我回去好好翻翻生态中心的研究队伍，「每个老师的实验室都是不同的，那里的氛围怎么样，就可以看出这个老师给人是什么感觉」；列个清单，她看看可不可以帮我推荐一下。 「熟人推荐」简直像是一根稻草，我虽不是溺水的旅人，但也忍不住激动地狂舞双手。然而，13页的老师名册中，我心仪的十位导师，却没有一个是H老师熟悉的名字。心情突然从天掉到了地，「稻草」断了。 一个一个地给老师发邮件，写尽了想加入他们的研究团队之类的话，结果却都是杳无音讯。 5月 >>充满鲜花的世界到底在哪里 我开始有放弃的念头了。 毕竟我只是一个普普通通的第一名，科研成果几乎为零，甚至因为种种原因，都拿不到一份国奖加持。面对生态中心，我大概是有「心」无力了吧。拿着两份暑期社会实践经历，转而去联系浙大和南大的老师。 很快，浙大的L老师把一封《欢迎加盟团队，做水稻生态系统效率方向》的邮件作为回复。我兴奋得一晚没睡，当下向L老师承诺，如果能进浙大的夏令营，我一定去他门下当学生。 那时的我，像找到了多年失散的家人一样，也像个孤儿被收养了一样。 「稻草」太容易断了。我痛恨这虚幻的、曾经、有可能的可能。我想要「栖身之地」，我想要被认可，我想要被给予一个机会。从高考失利到如今，从大一倒数的入学成绩到如今，从被迫进门到爱上这个专业，从三年前的我，到一年后的我。我想要中科院，但我也是真的感谢那封邮件，让我得以在那个五月活了过来，让我得以在之后的几个月里继续「折腾」。 月末，各高校夏令营开始报名，我信心满满报了南京大学，浙江大学，还有我割舍不下的生态中心。这一个月里我从未真正地「死心」，依然在不断联系、联系，再毫无音讯地，联系。 现在回想，那时真是一点后路都没留给自己。 6月 >>耿耿星河欲曙天？ 浙大，尚未审核通过。 南大，院系拒绝通过申请。 生态中心，审核中。 居然是生态中心现出了希望。 6月20日，生态中心夏令营名单终于出来了。120个人，我没有通过审核。 难道怪自己一语成谶，只能考了吗？把生态中心以往推免录的学生看了一遍，哪有「双非」起跑线的学生？6月23日， 中午一点五十——这个时间大概会被我记很久很久。不只因为我突然被吵醒，而是因为这通电话的归属地是北京，更因为对面的声音说："你好，Z同学吗，你被选中参加生态中心夏令营，请于明天前在网上确认是否参加。” 我简直飙泪。废话，可能不参加吗？但那时跃入脑海的，居然是另一个念头：这该不会是别人幸灾乐祸的恶作剧吧。 7月 >>天堂地狱 一线之隔 八方打听，听说武汉水生所只要进夏令营，全都可以有水生所的推免资格。于是便无忧无虑，玩了近半个月LOL，只可惜没从青铜爬出来。 7月16日，夏令营开营。对我来说却是打下一个晴天霹雳——生态中心的夏令营只是讲座，只是给学生一个提前认识老师、联系老师的机会。能推免到生态中心还是需要所内的博导推荐。看了下夏令营名单，一百二十四个人中，只有三个双非学生。 最后一天，导师见面会。招生办老师说：不可能招「双非」的推免生。 我的梦要破灭了。 8月 >>吾志所向，一往无前 夏令营结束后，我没有回家，而是回到了武汉植物园，学习查SCI文献。
