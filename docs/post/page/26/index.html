算不上请愿的请愿书

十月 26, 2018 发布在 未分类
算不上请愿的请愿书算不上请愿的请愿书 HJK 玩转湖大 写在正文前： 鄙人，12级软件工程，黄金坤，玩转湖大初代builder，本想当一只安静的考研狗，最近有件事让我没办法在幕后继续淡定下去了，恩，我正有写一点东西的必要了。 这次推送纯属个人举动，与组织无关。如有破坏校园和谐大一统氛围，一切后果本人自行承担。 正文标题：老鼠屎（shi），快放开我们的图书馆！ 早在多日以前图书馆大门就张贴了禁食令。无奈鼠灾肆虐，悲剧接二连三发生。每天早上甚至都可以听到工作人员在提醒进馆的同学禁食的相关条款，饭后门卫大叔的座位后更是满满地陈列着缴获的违禁食物。然并卵。。。老鼠似乎任然痴迷于啃书，不得不承认，它们确实比很多考研狗啃的刻苦。 于是学校放大招了。。。 “诶，各位考研的同学看看门口的通知” 1号下午，我照常的在三楼自习室复习，突然传来值班老师冰冷的声音。和平常偶尔工作人员的闲聊一样，似乎并不介意打破自习室应有的安静。 通告是这样的： 看到这个很多人内心是奔溃的、是拒绝的。考研圈顿时一片哗然，大家纷纷以各种方式表达着自己的心声。更有壮士第一时间用血和泪谱写感人的篇，在朋友圈广泛流传。 例如这位： （同学，开门，你的快递到了...） --------（分界线）-------抵制也好，愤慨也罢。 事情从来都是需要双方协作才能更好解决。 我们承认，这事跟我们有很大关系，食物是我们带进去的，少量童鞋并没有及时处理掉残渣留了口粮给老鼠。有些细心的同学可能注意到了，相当大的区域是很少有人打理需要自觉维护卫生。而且我们的书本的确摆放很随意、杂乱，在没有相关规章制度约束前，官方说我们阻碍消防卷闸门通道也是合情合理。 接受不能改变的，改变可以改变的。 我们暂时改变不了图书馆老鼠成患，那么从现在开始规范自身行为，不带食物进图书馆，我们可以做到。 但是要我们把全部书本撤回寝室，每天搬家一样的去图书馆，这个我做不到。 我希望和图书馆协商争取更好的解决方案，但是这需要大家的帮助和支持。

让我这个创始人说句话

十月 22, 2018 发布在 未分类
让我这个创始人说句话让我这个创始人说句话 方小达 玩转湖大 在这里先给大家道个歉 由于我当年创办玩转湖大的时候的疏忽，导致昨日图文未经审核被直接发布，给大家带来了困扰，同时也给现在负责维护玩转湖大的小伙伴造成了麻烦。 下面我先说说我昨天一天做了什么 在我早晨打开玩转湖大的时候，收到了《算不上请愿的请愿书》图文消息推送，我感到有些诧异，将HJK的图文发布权限删除之后，为了进一步了解事情的来龙去脉，我立即联系了作者HJK，没有联系上；另一方面，我向学校记者团了解相关信息，记者团告诉我，图书馆方面对于12月1日的告示已经接受过记者团的采访，采访原文如下：（已经征得记者团允许转载） 到了这一步，我发现，事情并不完全是HJK想的那样。 即使今天我还要上研究生课程，但是我还是不时打开微信后台，了解同学们对这件事情的看法，只可惜受到图文消息评论数量限制，无法将所有的评论发布出来，我只能尽力将各方的观点都发出来，尽可能做到公正客观。 下午，有同学通过后台告知，校学生会、校图书馆、考研代表也就这个问题开了一次会，校学生会官方微博上也发布了会议结果，具体如图所示：（这条微博的发布时间是12月3日） 看到了这个，我差不多知道了到底是怎么回事了，事情至少在朝着好的方向发展，各方都在努力让这个问题能够得到圆满解决。 果然，到了晚上，有同学告诉我图书馆门口又贴出了一张新的告示： 作为一个曾经在图书馆泡了四年的湖大学子，我为这样的结果感到欣慰，图书馆方面能够在如此短的时间内做出适当的反应，确是我湖之幸。 好了，镜头转向图文作者HJK 我知道他在准备考研，而且报的学校还是北大，面对不甚明朗的考研形势，心理压力有些大，在中午我终于联系上了他，简单寒暄几句过后，我把当时从校记者团那边了解到的图书馆的情况跟他做了一个说明，同时告诉他这件事上他太过意气用事了，虽然他在图文消息里边写着和其他任何组织或者个人无关，但是实际上还是影响了玩转湖大的形象，HJK也表示现在再看这条图文消息，个人主观倾向太过明显，就这样发布在玩转湖大上边，确实不妥，并且与我立下约定，在接下来的时间里，绝不做任何无关考研的事情（毕竟对他来说，考研才是目前最重要的事情，我也不想因此扰乱他备考节奏，同时，我相信他是一个能够遵守约定的人），等到考研结束之后再和他谈这件事。 至于考研结束后怎么谈这件事，我只是希望他能够借这件事情记住，心中有不满没问题，有想法是好事，但是一定要通过正确的方法来表达，而不是像这样直接发布图文消息，让所有人被他的情绪带动起来，让那些默默努力用实际行动真正解决这个问题的人徒增许多压力。倘若他真的能够理解，哪怕玩转湖大受此影响，我觉得是值得的。 要做有用的事情 玩转湖大经过这件事情之后，让我这个远在杭州的创始人又多了一些思考。我和HJK还有TR她们将玩转湖大交给现在的维护伙伴的时候，说过，要做有用的事情，至于什么是有用的事情，我们谁都说不清楚，只能够靠大家自己去探索。觉得讲座信息对大家有用，于是我们做了讲座信息；觉得学校活动对于大家有用，于是我们发布学校活动信息；觉得学校的一些通知对于大家有用，于是我们发布学校通知信息...玩转湖大从我创立之初便是如此。 记得当时，我把玩转湖大的查课表、查图书、查体测这些基本功能做完之后，面临着一个问题——我到底只是做一个有用的工具还是做一个有用的『人』。做一个有用的工具很简单，那些基本功能都可以算得上有用了，甚至包括现在这些讲座通知消息之类的，这些消息是没有生命的，仅仅是按照我的代码运行的『程序』罢了。 我想做出一个有用的『人』 是的，作为创始人，我更多的把玩转湖大当做一个『人』看待，虽然是我创造了他，但是他和我是平等的，他每天也在接收着外界各种信息，他也会形成自己的『想法』，形成自己的『态度』和『观点』，他会被人讨厌，也会被人喜欢。若要说做到完全中立和客观，那我倒不如写一段新闻自动抓取程序罢。 不论是被人讨厌还是被人喜欢，我都希望玩转湖大能够以合适的方式表达自己，同时把握自身的底线和原则不动摇，不同的人对于玩转湖大有着不同的期许，一千个湖大学子心中有一千个玩转湖大，但是，一旦玩转湖大作为『人』的形式而不是『工具』的形式存在，那么，玩转湖大就是玩转湖大本身。 我一直羡慕暴走大事件的王尼玛，因为大家把他当成『人』来看待。 关于昨天图文消息我的看法 说来也好笑，本来HJK图文消息是希望大家能够就图书馆搬书这件事情说说看法，结果不知怎么大家都扯到了占座上去了，事实上，文章全篇都没有提及半点『占座』的事情。这之间到底是怎么扯上关系的我觉得倒是挺有意思的。 也许是积怨已久了吧！ 正如大家所看到的，图书馆官方最后在搬书方面做出了让步，但同时严厉禁止占座行为。 林子大了，什么鸟都有。 我相信，即使学校措施再怎么严厉，占座、谈恋爱、吃东西、玩手机这些事情总会有漏网之鱼，这个时候就轮到我们发挥作用了。 作为一个泡过四年图书馆的人的经验，很多情况下只要我们主动上前提醒就好了，毕竟脸在这个时代还是很重要的，我遇到的绝大多数情况也都是这么解决的。 至于那一小撮无法解决的人，直接跟楼层大叔或者阿姨说吧，他们会处理的。
