老司机带你玩转湖大（第一辑）老司机带你玩转湖大（第一辑） Azkaban 老宿管 玩转湖大 午后整理扣扣，悠闲分类后 「有一半以上都是大学校友」 惊叹自己如此依赖校友资源。下午跑步回来，浑身汗涔涔的，却被小学妹揪着问问题，一口一个学姐喊的欢快。澡也不着急洗了，胡侃了起来。后来吧，慢慢发现 「大学，总不是孤军奋战。」一级带一级，是传统，更是高校存在的重要意义。于是决定--闲着嗑瓜子儿时，就把资源整理整理，跟大家互享，顺便八卦下学校小秘密和小发现~ 材料 上天--日常写作论文材料搜集 校图书馆官网电子文献库？必须连接校园网百度搜索论文？很大一部分需要付费  写作常用---国家图书馆。 国图主要可以用万方、维普以及中国大百科全书日常论文材料收集--作业材料、辩论赛材料... 只使用国图绝对够了。小贴士：国图注册只能用邮箱，手机号不行 。 以我的一次作业“公平贸易”为例： 入地--新闻与经历 论文太专业，只将论文当作参考资料写出来的文章，感觉飘在天上，我们还需要故事（通常为新闻和经验陈述）材料来源--知乎、谷歌写学术论文（毕业论文啦等）就要上知网，不过听说知网现在好贵的哦，北大都拒绝买了。 *信息网站（星号，选看）1. 中国社会科学网 ◈ 链接：http://www.cssn.cn/ 作业主题是什么，就查什么超级好用 2.199IT互联网数据中心 ◈ 链接：http://www.199it.com/ 3.纽约时报（又:新乡时报） ◈中文版 链接：http://cn.nytimes.com/  ◈英文版 链接：http://www.nytimes.com/ 可以当作知乎补充版，查类似“中越关系”这样的作业，会很好用. 4. 另外bbc news 也可以进啦，不过个人觉着纽约时报的内容好些。 练习英语的话可以进纽约客（the new yorker) ◈链接：http://www.newyorker.com/ 搜索china 看关于中国的新闻，连蒙带猜还是看得懂的。 2 连外网对着一块钱上毛主席发誓，我连外网是为了学英语。  推荐软件--蓝灯。搜索一下，会有软件下载。 下载后，在浏览器搜索wiki,fb,youtube,twitter,谷歌搜索，以及上面的链接都是可以顺利打开的。但是部分外网还是打不开，比方说“卫报” 谷歌搜索的使用和百度不太一样，为方便大家使用，就顺便贴个“快搜” ◈链接：http://so.chongbuluo.com/ 里面是搜索及信息网站大全 3 浏览器 谷歌浏览器，插件多，方便用（以后单开一个图文介绍几款插件）初期使用不会用的地方，还是要百度一下。

四月 20, 2016 发布在 未分类
读本书没你想的那么辛苦读本书没你想的那么辛苦 ZM 玩转湖大 有人说：每一场不经意间的相逢，都是命中注定， 在湖大，三生有幸，遇见天问。 在这里， 我们以书为马，览尽三山五岳到天涯； 在这里， 我们以书为雅，谈尽琴棋书画诗酒花； 在这里， 我们以书会友，就像拥有另一个家。好好好我说人话！我们的全称是：湖北大学天问学生创新团队。 一个纯粹凭兴趣参与的读书沙龙。 所以我们的另一个称呼叫“天问学术沙龙”。 以学术为志业，求独立之精神； 以读书为乐趣，存自由之思想。 是我们的理念。 我们的日常 1、田家炳教育学院206室是固定活动场所，每周六晚都在这里精读一本书的一部分，一个学期下来大概能读完一本书。 比如说……我们读《论语》比如说……我们读柏拉图的《会饮》再比如……我们读卢梭的《论人类不平等的起源》 当然除了读这些高大上的书，我们也要读其他的呀！ 2、大家商量好利用课余时间读小说，然后定个时间讨论，各抒己见。更重要的是……我们有萌萌哒的老师们参与啊！比如说，上周读的《不能承受的生命之轻》，接下来即将开读的《围城》3、我们的活动场所可不仅局限在教院的会议室里 Look！ ★图书馆茶室里，交流《万历十五年》朝夕青年空间里，和张爱玲相约《半生缘》沙湖边，谈论一番梭罗的《瓦尔登湖》图书馆门前的草坪上，聊聊《给樱桃以性别》 现在在读啥？以及怎么加入？ 1.我们现在每周六晚上读卢梭的《论人类不平等的起源》，英文版和中文版相结合来看，怎么样？不管你是政法院、文学院、历史系、生物系，外语系……人类起源总是与我们息息相关，博览群书应该是大学生们孜孜以求的正业，特别是还能提高英语水平呢！ 时间：每周六晚7点到9点 地点：田家炳教育学院206室 2、周五晚上，我们也有读书会，这个不是每周举行，隔一周举行一次，大家选取自己感兴趣的书籍进行带读。具体安排如下： 在这里， 既能学到知识、拓展视野， 又能结交到很多好玩的小伙伴。 在这里， 有天问内部成员、研究生学长学姐参与， 还有志趣相投的各专业老师。 在这里， 不了解的书你跟着读， 感兴趣的书你推荐读，擅长的书你带着读。 对天问有兴趣的话， 带上你的灵魂和肉体在周五周六的晚上约起来吧！ 记得加上QQ群了解最新的读书会信息及动态~ 1、天问学术沙龙QQ群：168036496 2、天问学术沙龙微信公众号： 诚邀志趣相投的朋友加入我们， 跟我们一起谈古论今、谈天说地！
