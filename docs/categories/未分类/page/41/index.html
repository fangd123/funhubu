商院志协带你吃喝玩乐---踏青好去处商院志协带你吃喝玩乐---踏青好去处 刘荣 玩转湖大 商院志协“我走过的武汉说出来才精彩”之风景篇春天来了，暖暖的阳光让人的心情也一下子开朗起来，商院志协为你推荐一些踏青好去处，接好咯哦~1梨园位于东湖风景区风景很不错，很多仿西式的小型建筑模型，很适合野炊或拍照取景，里面还有个屈原纪念馆，可以去看看。2木兰天池这个风景区山不是很高，但空气很好，很清新，还可以在那里宿营，晚上一起看星星看月亮的。3东湖磨山自然风光很美可以爬山，锻炼身体，里面也有许多花，春天，正是樱花桃花争奇斗艳的时候吧~4沙湖后面是农田，很有自然风光，重要的是离学校很近，可以结伴骑车去。沙湖公园也是个好地方，晚上去散散步也是极好的啦~5江滩自由浪漫，和闺蜜，男票女票拍照留恋的好地方呢~6蛇山沿着后山的马路走，一路风景很好，附近还有一些革命纪念馆可以去参观~7东湖海洋世界虽然武汉没有海，但仍然可以和海里的物种接触哦~8武汉长江大桥一桥飞架南北天堑变通途上层为公路，两侧有人行道，下层为复线铁路。据说情侣一起走过长江大桥就会幸福一辈子哦~9珞珈山位于武昌中部由十几个相连的小山组成，在山顶可以俯瞰东湖全景和武汉景色，现在有周恩来故居，郭沫若、郁达夫、蒋介石别墅，旁边就是武汉大学啦~10解放公园有供游人娱乐的电影院、网球场、游泳池和摩天轮、太空船、飞毯等现代化游乐项目，以及配套的饮食、茶社等商服网点，是市区最大的自然生态公园。 11落雁岛岛屿游乐世界，吟波楼”、“情侣楼”、“吹笛桥”等景点楚色楚香12长春观这里有最萌的小道姑哦~感受她的琴棋诗画~小编说~带你去你所想踏青虽好，可别忘了注意安全哦~ 还有戳“公交路线”便可带你去你所想哦~
