校园网（HUBU-WLAN）答玩湖记者问

五月 24, 2015 发布在 未分类
校园网（HUBU-WLAN）答玩湖记者问校园网（HUBU-WLAN）答玩湖记者问 丸子 玩转湖大 关于湖北大学校园网（HUBU-WALN）的使用问题，针对同学们关心的问题，小编特意咨询了湖北大学信息化建设与管理处，答疑如下：1、现在校园WiFi是免费使用吗？目前该无线网处于测试阶段，大家在测试阶段可以免费用，测试的结束时间待确定，到时候会通过校园网主页和湖北大学信息门户发布通知！账户是本人学号，密码是身份证上的出生年月日（8位数字）。过了测试阶段之后，校园网用户仍可使用，不需另外再缴纳费用，也就是说校园网用户只需要每个月缴纳25元，就可以同时使用寝室的校园网宽带和校园内无线网络覆盖区域的网络。2、校园网现在覆盖的区域有哪些？我校无线网建设启动于2015年3月22日，目前刚完成了一期建设，一期建设无线AP约1300个。覆盖区域包括：思睿广场、求索广场、大学生活动中心、综合楼（部分区域）、图书馆、学院办公区域（部分）、校医院、食堂和阳逻校区办公区、教学区、教学楼前广场以及食堂等区域。3、有同学反映教学楼和图书馆上个厕所回来就断网了，这是什么原因呢？（1）因为资金限制，作为一期工程的校园无线的覆盖范围有限。就目前来说，湖北大学图书馆的wifi是实现了全覆盖，逸夫人文楼的图书室也有wifi覆盖，此外，各个学院的办公室以及会议室都可以接收到无线，各教学楼目前只有一楼至三楼有无线覆盖。所以有部分同学反映的在较高楼层几乎搜索不到信号，是因为目前学校并没有在高楼层安装无线WIF发射和接收装置呢！当然，如果你运气好，在四楼教室也是可能搜索到hubu-wlan的哦，这种情况下当然信号会很弱。如果你喜欢体育锻炼，二号体育馆也是有wifi的哦。大学生活动中心的wifi也是少不了的，排练节目，休息空闲的时间都可以享受到无线的便利。学校的三个食堂，包括风华园，嘉惠园，以及桃李园都可以搜索到无线信号，一边吃饭一边上网，听起来是不是很惬意呢。（2）使用无线WIFI时，认证页面不能关闭，需要处在打开状态，当然你可以将其缩小或放在后台。因为一旦关闭，系统会认为用户下线。另外，如果你在登陆后的30分钟内不进行任何操作，系统是会自动将你踢下线的哦。4、您还有什么叮嘱同学们吗？有，最后还要公布一个不知道是不是坏消息的消息，根据学校的安排，教学楼的wifi会在早上八点到晚上六点期间进行关闭，同学们要安心学习哦~小编说几句今天总算是把关于HUBU-WLAN的问题给大家问清楚了，除了感慨覆盖范围好广，小编我终于可以发朋友圈，炫耀我湖遍布各地的WiFi信号了！！！关于校园网还有什么疑问可以点击下方“阅读原文”进入微社区留下你的问题哦~小编火速赶来为你答疑解惑。阅读原文
