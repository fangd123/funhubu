2017“校园邮乐场”活力来袭2017“校园邮乐场”活力来袭 玩湖 玩转湖大 11月2日，中国邮政2017年“校园邮乐场”品牌推广活动将入驻湖北大学。本次活动以“未来‘邮’我创”为主题，线下活动正在走入全国12座城市的60所高校。 “校园邮乐场”活动为了紧密贴合大学生群体，在活动现场利用场景化和模块化的创意集装箱，打造现代感的场景体验区，巧妙融入邮政发展概况及邮务、金融、寄递、电商等业务板块，并搭载了戊戌年狗票创意合影活动、明信片设计大赛，邮储银行享乐学活动等丰富内容。让大家在趣味互动过程中认知邮政发展现状。 活动现场以绿色与黄色为主色调，散发着青春无限的阳光味道，更有帅气的主持人邀你体验丰富的特色活动。在这里，既可以抢鲜看到最新的生肖邮票，跟可爱的狗狗邮票合影许下“新年我要旺”的美好愿望，还可以了解到邮储银行针对年青一代开发的青年卡，享受海量优惠活动。现场还能找到邮乐网特别打造的“家乡的味道”专区，带每一位吃货重回舌尖上的家乡，更有“邮乐小店”助你不出校园就能自主创业，轻轻松松赚取零花钱。 现场还有欢快的趣味小游戏等你参与，在游戏中不但可以快速涨知识了解中国邮政的发展状况，还有机会获得如超薄笔记本电脑、电子阅读器、智能运动手环、水杯、充电宝等精美奖品。 今年除了既有的“邮政进校园”路演活动外，还特别新增了“学生进企业”的“邮政体验日”活动，将在北京、上海、南京、武汉、广州、成都、西安7座城市共计举行9期活动。通过线上的体验日活动报名，就会有机会深入了解中国邮政的历史，能够参观展现邮政发展成果的典型场所，可以与邮政专家面对面互动交流，甚至还有可能获得中国邮政校园招聘的笔试机会。长按二维码报名参加有惊喜！ 中国邮政“校园邮乐场”系列活动，旨在增强大学生对于邮政多元化业务发展的认知，进一步强化中国邮政母品牌多元化、现代化的品牌形象，持续提升邮政在青年群体中的影响力。“校园邮乐场”未来邮我创，这个极具创想的邮乐场现在来到了你的学校，快来一探究竟吧！ 点击“阅读原文”，一起加入这股朝气蓬勃的青春浪潮吧！
