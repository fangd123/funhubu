GRE普通考试—出国读研的最佳途径

十二月 27, 2017 发布在 未分类
GRE普通考试—出国读研的最佳途径GRE普通考试—出国读研的最佳途径 木木 玩转湖大 考虑毕业后出国读研？ 这是一个非常明智的决定。与其在国内考研，将就业压力延后几年，越来越多的同学选择留学，以追求：国外高薪实习和工作机会、提升回国后就业竞争力等。 作为全球最大经济体，美国名校云集，也一直是广大同学的留学首选目的地。美国社会商业氛围浓厚，却极其重视教育和人才，为留学生提供了慷概的奖学金和助学金。美国还为优秀的留学生提供了大量实习和就业机会。不过，赴美留学不得不提一个重要的敲门砖：GRE考试。 GRE（美国研究生院及商学院入学考试）是全球认可度最高的研究生入学考试，也是唯一一项研究生院和商学院都认可的学术能力考试。GRE成绩是众多院校筛选学生、授予奖学金时采用的金标准。 GRE主要考察：文字推理（语文）、数量推理（数学）、以及分析性写作（作文）等学术能力。这些学术能力对于硕士、博士阶段的学习和科研至关重要。因此，国内外众多研究生院和商学院纷纷要求申请者提交GRE成绩，借此评估不同学生的潜质，预测他们在未来学业中的表现。有些高校的研究生院和商学院将GRE成绩作为第一轮初审时“一刀切”的指标，有时甚至作为唯一指标。 在众多申请的国际学生中，GRE成绩越高越有竞争力。申请名校的同学最好能达到325以上的高分。GRE分数越高，录取的几率也越大，拿到奖学金甚至全奖的几率也越大。例如被哈佛等7所名校博士录取的女学霸王若溪，在介绍自己成功经验中就指出，除了优异的本硕成绩及实验项目，GRE成绩与最终的录取也有密切关系。 全球认可度最广的研究生阶段入学考试 不管考虑申请研究生院还是商学院，或者还没想好今后攻读什么专业，你都可以尽早准备和报考，未来5年内无论决定申请哪所学校、攻读哪个专业，都可以使用。GRE是全球认可度最广的研究生阶段入学考试，因此GRE考生在院校和专业上就有更多选择，包括：理工科、人文社科、商科、法学各专业的硕士、博士、MBA等项目。 除人文社科和理工科的硕士、博士项目外，全球约1,300家商学院的MBA项目接受GRE成绩，几乎覆盖所有中国学生最常去的学校。例如，根据BloombergBusinessweek（彭博商业周刊）的排名，美国前30的MBA项目全部接受GRE成绩。除了常规的MBA项目和商科专业硕士外，商科专业的双学位、硕博连读等项目，一般只接受GRE成绩。商科与其他专业的交叉学科（金融工程、商业分析、数据科学等），往往要求学生提供的入学考试成绩也是GRE。对于一些新兴专业，有的大学设在研究生院，另一些大学则放在商学院，很多申请者搞不清楚应该提交哪项入学考试的成绩。这时应该首选GRE，因为GRE成绩认可度最广，全面覆盖研究生院及商学院下设的理工科、人文社科、商科各专业。此外，同学们千万不要以为“管理”都是商科专业。事实上，人力资源管理、项目管理、供应链管理等专业在国外大学分别归于人文社科和理工科，归属于研究生院，同学们只能用GRE申请。另外，MPA（公共管理硕士）虽然与MBA之有一个字母之隔，通常也归在研究生院而不属于商学院，也只能用GRE申请。 GRE方便考生刷分，仅报送最好成绩 在送分时，GRE考生可自主选择最高的一组成绩寄送给院校，申请院校只会看到考生寄送的这次成绩，而不会看到考生多次参加考试的历史记录。根据ETS的官方数据：“大多数重考的考生，成绩比首次考试有所提高，尤其是在语文和数学两个科目。”所以，同学们如果某次考试发挥不好，就可以复习一段时间，再考一次，直到取得竞争力强的分数。 不同学校、不同学科对GRE的要求是不一样的。如果考生的目标是全额奖学金或是牛校，则需要各项硬件材料都足够优秀，才有胜出的机会，GRE成绩作为学术能力及英语能力的双面检测标准，更是不能忽略。 分数有效期方面，GRE成绩有效期长达5年，如果考生在大一大二参加GRE考试并取得了不错的成绩（例如325分以上），大四上学期申请时成绩还有效，既可以申请本科同专业的研究生，也可以跨学科申请其他专业的研究生。即便毕业前改变主意选择就业，GRE成绩写在简历上也有助于增强就业竞争力，向雇主证明该学生能熟练运用英语进行高级阅读和写作。 报名方式 基本上每个月都有GRE考试。全国有约100个考点，考生可以自行选择。尽早报名，就能优先挑选自己意向的考点和考试日期。GRE需在线报名，请访问GRE官网：ets.org/gre/register。首先，你需要注册一个GRE账号（https://www.ets.org/gre/revised_general/register/your_ets_account），以便考后查看正式成绩。注册完成后，你就能报名了。如选择在中国大陆参加考试，之后会跳回：“教育部考试中心GRE网上报名”（https://gre.etest.net.cn/）网站。如有疑问，可拨打咨询热线：010-82345673。更多精彩内容，尽在阅读原文

新一代中国纪录片标志性人物周兵老师来啦新一代中国纪录片标志性人物周兵老师来啦 沐酉咖啡 玩转湖大 在前段时间里我们邀请了著名文化评论人、影视编剧、媒体人多角色集于一身的李亦燃老师做客沐酉（友），在短短的两个小时里，李老师为我们打开了一扇关于纪录片的新大门，原来纪录片可以是很有趣的。而这次我们邀请了，新一代中国纪录片的标志性人物之一周兵老师。 关于周兵 著名纪录片导演，现担任中央新影集团周兵导演工作室艺术总监，新一代中国纪录片的标志性人物之一。曾在中央电视台工作20年，曾任中央电视台《东方之子》栏目编导、《东方时空》特别节目组制片人、参与创办《东方时空·纪事》栏目，曾任中央新影集团特别节目部主任。三度获得中国纪录片年度导演。其创作及监制纪录片上百部，多次获得国内外纪录片大奖，代表作有《故宫》《敦煌》《梅兰芳》《千年菩提路》《下南洋》《外滩》《当卢浮宫遇见紫禁城》《雪豹》《汽车百年》等。 对于纪录片而言，我觉得周兵老师是在用生命拍摄，为我们讲述着这些历史的所带给我们的不为人知的事情，以及它们所拥有的魅力。 而在这次活动中我们将听听周兵老师分享这一路来的点点滴滴，让悻悻学子们向着自己的目标地前行不息。 一夕故宫筑就，百年风雨流传。 一曲长歌赋敦煌 “如果不是喜欢这里， 我不会来； 如果不是喜欢这里， 我来了也会走。” 千年国医 “一部讲述中医真实历史的纪录片；一次启迪生命的身心旅程；一个个激荡人心的故事；一部用心血倾注的精良作品。” 活动时间：12月23日（周六） 14:30 活动费用：免费活动人数：30人 活动地点：武汉沐酉咖啡（沙湖路巴厘龙虾旁50米，铂金天地小区对面） 报名方式： Ⅰ.添加微信 MUCafeStaff 询问详情.
