我怀念的是2013…吗？

十月 18, 2018 发布在 未分类
我怀念的是2013…吗？我怀念的是2013…吗？ 原创：李安 玩转湖大 定制故事002真实性：50% 故事人：2014级 姜可（化名） 来自主人公的话：我希望，我和你的感情，是不会散场的老友记。 冬天是个残忍的季节，尤其是武汉的冬天。沾了雨的冷风一吹，上下牙关忍不住一齐跳舞，直想逃离这座浩然地狱。 姜可第n次拿起手机，屏幕依然一片平静。太冷的时候，人心也会冷的很快。她终于忍不住了：「我们分吧。」 迷迷茫茫中，她又发了不少信息给他。就像是扔一颗又一颗失了引信的手榴弹——她怎么可能忍心伤害他，她只是气，又不知何处示弱。满心疮痍，每一处洞口都是哪些手榴弹曾经呆过的地方。她再不扔去，就要在心里爆炸了。 人长大了以后，眼泪就少多了。从那么某一年开始，连接泪腺的通道，已经拐弯回转向心里。现在，她的心就被眼泪浇得湿淋淋凉飕飕的，睡觉也不那么踏实，辗转半宿都在迷蒙间徘徊。脑子里一片乱想，也分不清究竟是梦还是回忆。 大抵不是梦吧，梦怎么会这么明亮。一片金澄。 所以，我是真的回到了2013？ 姜可不敢想象，但课桌那边的确是货真价实的他，温柔地笑着，双瞳荧荧，就像是两颗玻璃质地的葡萄那么好看。 2013，是她说「我们坐同桌吧」的那个年份。 高三下，鉴于对自己惨淡数理化的正确认知，她向他提出：「我们互相监督，每周日一起来学习，平时不要玩手机。我帮你英语，你辅导我数理化。」 同样的一道题，她眼里是一团迷雾，在他手里就变成了柳暗花明；更奇怪的是，别的“学霸”偶尔也会给她讲题，却都不比他讲的清楚透彻。但他也有自己的小毛病：容易在一道题目上死磕——遇到之前没见过的形式，就会不相信自己算出的结果，转而要写一个记忆中的错误答案。 真是服了他这点。 刚坐同桌的时候，她莫名的胃痛。每天吃不下东西，甚至有时还会反胃。校医没检查出来个所以然，配的药也毫无效果。她只能强用「多喝热水」来对抗，实在忍不了了，就用力掐自己的虎口。 他看到之后，默默拎了一壶热水放在座位中间，天天如此。 过了一段时间，胃终于恢复了。他知道之后尤为兴奋：「想吃什么好吃的，我这周末可以给你做了带来。」她以为他只是随便说说，也痛快回复：「想吃超好吃的蛋炒饭～」 没想到周日早上，他真提来个保温桶，说：「饭来咯。」她完全呆住，他却一脸平静，边掀盖子边说：「不是你说想吃嘛，来来来，尝尝本大厨早起半小时的手艺。」金色的米饭配着细碎的鸡蛋，油盐正好，没有半分打扰葱花的淡淡香气，系数任它们萦绕在旁边的鸡翅上。面前一片诱人的金黄，配着他递来的豆浆，好吃得让人恨不得把勺子都吞下去，连他在一旁碎碎念的烹饪过程都没注意。吃出半分饱意后，才突然想到他，最后恋恋不舍地看了一眼蛋炒饭，便狠心推给他：「你要是不嫌弃，就别再去买了……这太多了，我吃不完。」 心里莫名的开心填饱了她未满足的胃。 时间不知怎么，转瞬就到了夏天。离高考只剩不到三十天，大家都开始拼尽全力了。和他坐一起之后，姜可的成绩有了很大提升。反倒是他，每次都不大尽如人意。 周六，他提着一个黑色的袋子走进教室，向她示意：「陪我喝点去。」姜可猜，大概是这次的成绩又让他难以满意，就跟着他一起去了顶楼。打开袋子，她才发现他带的是白酒。「你要是喝不来就别喝啊。」他稍稍拧了一下眉毛。姜可拿过酒瓶：「才42度，不算什么。」小时候在家喝过几次，她心里还算有底。 一口酒，拌几句学习考试之类的话。没多久，姜可率先喝完，看着他：「你一口喝完剩下的，不许再喝矿泉水。”他连摆手：「不行不行，真喝不来。」姜可半醉半醒看着他，他眼睛真好看，黑亮黑亮的，像是盛了整个月亮进去。他又突然开口：「我喜欢你。「她笑着打了个嗝：「那是，我也喜欢你。」他坐立起来，表情严肃了一点：「如果，大学你没有男朋友，我们就在一起吧。」她不知他还剩几分清醒，依然笑着说：「那就等大学毕业吧。要是都单身，就在一起。」他迷迷糊糊「嗯」了一声，似乎快要睡着了。 第二天，他问姜可他有没有说什么奇怪的话。为了避免尴尬，她什么都没说。 高考结束，暑假开始。姜可家里原有的温馨，被「出分」折磨得紧张兮兮。唯有每天和他聊天的那段时间，他简直分秒都当百忧解。越聊越多，越聊越晚，不由自主地，她突然冒出来一句：「我们在一起吧。」 他很快回复：「好啊。」 从「好」字开始，到「啊」字升华，刹那间升起的紧张让姜可几乎按不准打字键盘。明明之前还那么镇定，她忍不住开始嘲笑自己。但很快就把话题转到了以后的计划上：志愿填什么，去武汉一起玩什么，甚至都说到了以后要把家布置成什么样……天马行空地乱想，忍不住笑了出声。 但现实从来不按脚本拍摄。 23号一早，姜可先查，成绩好于预期；他那边却杳无音讯。几乎熬过了她人生中最漫长的一个多小时，他回了一句：对不起。就又消失不见。她下了最后通牒：「再不回复，我就不理你了。」 他才吞吞吐吐说了成绩。低到她完全懵了。却是他反过来安慰姜可：「没事，我决定了，再读一年，去武汉找你。」 日子如流水，分隔两地也让两人逐渐习惯了隔着电波的交流——放学他不坐公交回家，走回去的路上和姜可讲电话，说说一天发生了什么好玩的事。 出分那天，她五点多就醒来。辰光在天际线上跳来跳去，百般纠结后跳转的页面让她再次如陷冰窟。 七点多，他发信息给她：我知道成绩了，会尽力去武汉。 最终，他去了离武汉只有几十分钟车程的城市，开始了他的大学生活。 她终于从梦境中脱身，睁眼看着漆黑的寝室。 从金色开始，到黑色结束。从2013开始，又落回2016。落回他忙得马不停蹄的这个学期——专业课变多，开始在辩论队带新生，落回陪她的时间越来越少的这个学期，落回他们已经在一起900天的现实中。 她几乎要流下泪来。我好怀念2013。那个散发着香甜蛋炒饭味道的年份，那个一偏头就看到他的年份，那个我们都没有说爱却很温柔的年份。那个我已经无法回去的年份。 但时间不可追往，亦无处溯返。 大学是一道清晰的分水岭，每个人，每件事都在和之前的样子渐渐划开距离，或多或少地变化着。姜可看到了他回复的信息：「高三高四只有学习只有你，现在还有兴趣爱好，要努力提高自己，将来还有事业，还要照顾父母……我想要力图改变，让自己更加优秀。」 她百感交集。我懂，我都懂，我也会有忙到昏天黑地。如果说朝夕相处如同葱花，是细细碎碎的快乐；那么异地相恋则似鸡蛋，大概得要一番精心烹炒才见得其香吧。 然而……她的「然而」突然被堵在胸腔——是他的第二条信息：「我还是一如既往地喜欢你。」 其实2013，和2016，只不过是两个年份而已。我们都长大了，我们隔了好远，我们有了和对方不再完全重合的人生轨迹。因为诸多不可抗力，时间、地点、形式统统都不一样了。 但最重要的我们，还是像2013年一样，没有分开，依然相爱。 这就够了，我想和你好好的。 度过很多很多个日月年光。 题图摄影：Michael 所以，我在中科院了。

十月 18, 2018 发布在 未分类
所以，我在中科院了。所以，我在中科院了。 原创：周思宁+李安 玩转湖大 定制故事001真实性：99.9% 故事人：2013级 资环学院 周思宁 来自主人公的话：感谢黄老师的教导，感谢大学给予的挫折让我成长让我强大。感谢中科院的机会，最重要的是，谢谢一路上有你❤ 所有关于「学霸」故事的开头，似乎都是他们并不认为自己是「学霸」。非常不幸，我也算是其中之一。不谦虚地说，今年四月我已经在思考准备保研何处的事情了。大一大二都是班上第一，大三上的绩点是94.13，湖大本校保研的似乎够了。然而继续坐在这口「井」里，像已经度过的、没有多少讲座、没有多少专业交流的平淡三年一样的日子，我已经，受够了。在网上反复地查了又查，还可以试试浙大和南大。 有条评论突然跳进我的眼帘「中科院生态中心是排专业第一的吧。」 好像有什么东西怦然炸开了。炸在我心里。 大二下学期，我去了武汉植物园做实习生。所以某种意义上来说，从那时起，我就加入中科院了。然而在我们院的考研史上，只有一位04级的学长真正试过。 生态中心只回复了一封邮件：不招收双非院校学生。 说实在话，我们院考研的，不乏清华浙大之类的好学校。我那时就想，能考上的我都不报了，要报就报最好的。于是毅然，非生态中心不去。 4月 >>「稻草」何处来？ 跟我的植物园导师H老师说了我的考研念头，她让我回去好好翻翻生态中心的研究队伍，「每个老师的实验室都是不同的，那里的氛围怎么样，就可以看出这个老师给人是什么感觉」；列个清单，她看看可不可以帮我推荐一下。 「熟人推荐」简直像是一根稻草，我虽不是溺水的旅人，但也忍不住激动地狂舞双手。然而，13页的老师名册中，我心仪的十位导师，却没有一个是H老师熟悉的名字。心情突然从天掉到了地，「稻草」断了。 一个一个地给老师发邮件，写尽了想加入他们的研究团队之类的话，结果却都是杳无音讯。 5月 >>充满鲜花的世界到底在哪里 我开始有放弃的念头了。 毕竟我只是一个普普通通的第一名，科研成果几乎为零，甚至因为种种原因，都拿不到一份国奖加持。面对生态中心，我大概是有「心」无力了吧。拿着两份暑期社会实践经历，转而去联系浙大和南大的老师。 很快，浙大的L老师把一封《欢迎加盟团队，做水稻生态系统效率方向》的邮件作为回复。我兴奋得一晚没睡，当下向L老师承诺，如果能进浙大的夏令营，我一定去他门下当学生。 那时的我，像找到了多年失散的家人一样，也像个孤儿被收养了一样。 「稻草」太容易断了。我痛恨这虚幻的、曾经、有可能的可能。我想要「栖身之地」，我想要被认可，我想要被给予一个机会。从高考失利到如今，从大一倒数的入学成绩到如今，从被迫进门到爱上这个专业，从三年前的我，到一年后的我。我想要中科院，但我也是真的感谢那封邮件，让我得以在那个五月活了过来，让我得以在之后的几个月里继续「折腾」。 月末，各高校夏令营开始报名，我信心满满报了南京大学，浙江大学，还有我割舍不下的生态中心。这一个月里我从未真正地「死心」，依然在不断联系、联系，再毫无音讯地，联系。 现在回想，那时真是一点后路都没留给自己。 6月 >>耿耿星河欲曙天？ 浙大，尚未审核通过。 南大，院系拒绝通过申请。 生态中心，审核中。 居然是生态中心现出了希望。 6月20日，生态中心夏令营名单终于出来了。120个人，我没有通过审核。 难道怪自己一语成谶，只能考了吗？把生态中心以往推免录的学生看了一遍，哪有「双非」起跑线的学生？6月23日， 中午一点五十——这个时间大概会被我记很久很久。不只因为我突然被吵醒，而是因为这通电话的归属地是北京，更因为对面的声音说："你好，Z同学吗，你被选中参加生态中心夏令营，请于明天前在网上确认是否参加。” 我简直飙泪。废话，可能不参加吗？但那时跃入脑海的，居然是另一个念头：这该不会是别人幸灾乐祸的恶作剧吧。 7月 >>天堂地狱 一线之隔 八方打听，听说武汉水生所只要进夏令营，全都可以有水生所的推免资格。于是便无忧无虑，玩了近半个月LOL，只可惜没从青铜爬出来。 7月16日，夏令营开营。对我来说却是打下一个晴天霹雳——生态中心的夏令营只是讲座，只是给学生一个提前认识老师、联系老师的机会。能推免到生态中心还是需要所内的博导推荐。看了下夏令营名单，一百二十四个人中，只有三个双非学生。 最后一天，导师见面会。招生办老师说：不可能招「双非」的推免生。 我的梦要破灭了。 8月 >>吾志所向，一往无前 夏令营结束后，我没有回家，而是回到了武汉植物园，学习查SCI文献。 不是说只要有个博导推荐就行么，我不信如果有推荐，招生办还能直接删了我的「双非」信息不成？ 一息尚存，此志不容少懈。我准备给P老师发我的读书笔记，展示我的决心。H老师表示了肯定，也提出可以帮我修改。 我打印了300多页的文献，拿在手里意外有种踏实感。第一篇SCI，我看了整整四天四夜。发给P老师的第一篇读书笔记，不足一千字，但浓缩了两个星期的不眠不休。 没有等到P老师的回复。 却收到了朋友的信息：「我拿到了南开的保研资格，你也加油。」然而我要去哪儿加油？向学校导师询问的邮件迟迟得不到回复，身边也不缺冷嘲热讽的声音。只有H老师依然在支持我，鼓励我。我想，可能我是湖大的「弃儿」吧，只剩植物园还把我当「宠儿」对待。 那就继续咯。 经历过两星期的「地狱模式」，很快，我的阅读速度已经到了一天两篇的程度。发去第二篇读书笔记的后一天，成了我三个月来最开心的一天：P老师回复了！ 9月 >>一波三折 就在我以为P老师看中我之后，第三篇、第四篇读书笔记却又是再无回复。 焦头烂额中，我找了武汉植物园的两位老师，忧心忡忡交流了一上午，决定直接去约见P老师面谈。
