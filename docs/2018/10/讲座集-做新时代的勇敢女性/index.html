求学、治学与教学
陈关荣

时间：5月23日（周二）晚上7:00—9:00 

地点：华中科技大学一号楼学术报告厅

简介：

陈关荣 ，欧洲科学院院士，发展中国家科学院院士。陈关荣教授1981年获广州中山大学计算数学硕士学位，1987年获美国Texas A&M 大学应用数学博士学位，其后在美国Rice大学和Houston大学任教。研究工作主要集中在电子工程方面的一个核心领域——非线性系统的控制理论和动力学分析，及其在复杂网络等相关领域中的应用。

儒家的人格理想及其现代意义
李春青

时间：5月23日（周二）晚上7:00

地点：华中科技大学东九楼C103

简介：

李春青，北京师范大学文学院教授、博士生导师、兼任中国中外文学理论学会副会长、中国文学理论学会副会长、中国文艺批评家协会理事等。《诗与意识形态》等著作分别获得北京市人文社会科学优秀成果奖（两个一等奖，一个二等奖）、教育部人文社会科学优秀成果奖（二等奖、三等奖各一项）。


一带一路战略与经济史研究
陈争平

时间：5月23日（周二）晚上7:00

地点：中南财经大学经济学院会议室（文泉南206）

简介：

陈争平，经济学博士，教授，博士生导师，清华大学荣休教授、山东大学一级教授、中南财经政法大学“文澜”特聘讲座教授。



知识产权法律冲突的司法实务
陈荣传

时间：5月24日（周三）上午9：00—12：00

地点：中南财经大学文治楼6楼会议室

简介：

陈荣传，台北大学终身荣誉特聘教授


Finite Morse index solutions of the Allen-Cahn equation
王克磊

时间：5月24号(周三)下午3:00—4：00

地点：中南财经大学文波楼4楼会议室

简介：

王克磊，现任职武汉大学副教授， 2010在中国科学院数学与系统科学研究院获得理学博士学位；2010.7-2012.7在澳大利亚悉尼大学师从澳大利亚科学院院士E. N. Dancer 教授从事博士后研究；现主要研究椭圆与抛物型偏微分方程，特别关注于偏微分方程的奇异扰动理论及相关的自由边界问题等。


孟子说“勇敢”
蒋方

时间：5月25日（周四）晚上7:30

地点：中国地质大学教三楼109

简介：

蒋方，湖北大学文学院教授，湖北省优秀教师。长期从事中国古代文学与文化的研究与教学。出版《汉月边关万古情——昭君与昭君文化》《中国文化史九绎》《黄庭坚精选集》等专著十余部。她开堂讲座，观点独具，言辞间总能直逼人心，带着批判和质疑；她简单真实，待人如己，总有种温情在悄然间弥散，点染身边人。一位研究 《诗经》、《楚辞》的学者教授，一位长年浸润于古典诗词中的女子。

书香识女人——《红楼梦》中的女性
潘知常

时间：5月26日(周五)晚上7:00

地点：华中科技大学东九楼B101

简介：

潘知常，著名美学家、著名红学家(《红楼梦》)、著名战略咨询与策划专家。南京大学新闻传播学院教授，博士生导师，获国务院特殊津贴专家称号。中国民主同盟第八届中央委员会委员，历仼澳门比较文化与美学学会会长，《美学与文艺学研究》副主编， 中华美学学会理事，中国文艺理论学会理事，中国广播受众研究会理事，中华美学学会青年美学研究会副会长，中国摄影学会理论专业委员会副主任。曾获全国图书“金钥匙”奖。


我的爱永没有改变——电影《哈姆雷特》赏析
潘知常

时间：5月27日(周六)晚上14：30—16：30

地点：武汉图书馆

简介：

潘知常，南京大学新闻传播学院教授、博士导师，兼任澳门科技大学人文艺术学院副院长、教授、博士生导师。
