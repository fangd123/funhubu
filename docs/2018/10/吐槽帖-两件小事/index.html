原创： 方小达 玩转湖大

吾向来是以猜度别人为耻的，余以为每个人虽有阴暗面，但吾自信可以避开，至少不会因此伤害自身。然，两件事，吾按捺不住内心的不解，不知吾错在哪里，遂讲之。


吐槽帖~~~不喜勿看~~~


~~~下面开始说人话~~~


1. 成绩查询页面



这段话看得我一脸懵逼

黑人问号脸？？？

然后看到“维护绿色上网环境”，吓得我都变绿了，真是厉害了我的哥。


好吧，我承认，我放了微信收款二维码上去，还把我的守望屁股账号也放上去了，但它们都在页面最底端啊，属于爱看不看的那种，还有页面哪里诱导分享、诱导关注了？

这种破页面，本身只是方便大家查成绩用的，真的要做成可以分享那种，嗯，就差一个做前端的了，更别提诱导了，我自己都不好意思拿出去见人啊喂。

至于，诱导关注（玩转湖大？OW？），到现在为止，竟然没有一个人加我的OW账号（泪目~），这尼玛要是算诱导的话，真心还不如我聊妹子加的微信号来的多。

所以，这到底是who强行举报一波的？

我承认，查成绩的页面做的确实比较烂，还经常宕机（这个锅强行让我背我就忍气吞声背一下好了，给教务处留点面子），但是不爽可以直接和我说嘛，举报多没意思，联系方式公众号里边全都有，动动手指就可以和我说话，你说的在理，我肯定会虚心接受，如果是个妹子，兴许我还会给一些“福利”之类的表示感谢~蜜汁微笑 : )

2. 湖北大学微信号hubuxcb


前方高能，非战斗人员请迅速撤离~~
重复重复~前方高能，非战斗人员请迅速撤离~~

很久很久以前，我还在湖北大学党委宣传部勤工俭学的时候，那里的老师和同学（时任“湖大在线”站长WXH）都对我很好，可惜，我加入的时候已经大三了，技艺不精的我并没有为这个组织贡献多少力量就退了下来。

后来，我苦练技艺，终于在大四下学期开发了“玩转湖大”，我知道，我也只能通过这种方式来回报这所学校里曾经对我好、帮助过我的人了。

离开学校那天，我关注了所有的和湖北大学相关的公众号，怕的是漏掉点什么，不管是好的坏的，我都希望自己能够在第一时间了解，甚至在必要的时候，通过“玩转湖大”让更多的人知道这所学校每天发生的故事。

我不知道这是一种怎样的情感，也不知道这种情感能够持续多久，只希望，在我还有这种情感的时候，做点什么，让自己——心安。

我一直深信，这种情感是可以传递的，通过我的努力去传递，让这种感觉充满在我们这些毕业生之中。

然，胜地不常，盛筵难再；兰亭已矣，梓泽丘墟。只叹“物是人非”。

不知从什么时候开始，我的微信公众号界面就变成了这样的了。


湖大首页



历史消息
看到的是这个




明明已经follow了
收不到任何消息


百思不得其解，最后，所有的迹象都表明——

我被拉黑了。




在微信公众平台“用户管理”中将关注用户（粉丝）添加到黑名单分组后，黑名单中的关注用户（粉丝）将无法收到该公众帐号的群发消息、及自动回复消息，将粉丝加入黑名单分组后，粉丝无法接收到该公众帐号的群发消息，但可以通过“查看历史消息”查阅10条历史消息。


起初，我以为是微信坏掉了，把公众号取关然后重新关注了好几遍，结果还是那个样子，问其他同学（三届不同的），他们都正常收到推送。我作为公众号的运营者，深知做到“拉黑”这个地步，需要多大的勇气，或者说有多大的incentive。然而，计算机是不会撒谎的，事实摆在眼前，我认怂。


离开学校两年，当年认识的学弟学妹们大多也毕业了，面对事实，我无从得知到底发生了什么，只是觉得，这样，真的，不好。


高能结束。


我只想对那个在后台拉黑我的同学说：
