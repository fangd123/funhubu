他是湖大最受欢迎的历史老师，今天他来解析《美国政治传统及其缔造者》

十月 19, 2018 发布在 未分类
他是湖大最受欢迎的历史老师，今天他来解析《美国政治传统及其缔造者》他是湖大最受欢迎的历史老师，今天他来解析《美国政治传统及其缔造者》 朝夕青年空间 玩转湖大 美国政治，对于中国人来说既熟悉又陌生。可能是课本里的《独立宣言》、三权分立的历史词汇，印象里华盛顿建国、林肯解放黑奴的光辉形象，或者是需要批判的“资产阶级金钱政治”；或许是在网络上争论马里兰空气是否“香甜”、谈论《纸牌屋》与特朗普。 即使每天关于美国的政治新闻萦绕耳畔，我们似乎依然要问美国政治到底是怎么一回事？它是如何形成的？ 信息化的时代里，答案也许随手可得，但在经过真正的思考之前，我们仍然需要读书。这部《美国政治传统及其缔造者》写作始于二战中的1943年，当时霍夫施塔特尚为27岁的年轻人，然而这部作品却散发着成熟的气息，出版后影响力经久不衰，成为了解和研究美国政治的必读佳作。 「 作者及作品简介 」Introduction to the writer and works 理查德·霍夫施塔特（1916-1970），美国历史学家，哥伦比亚大学教授，1950-1960年代公共知识分子。代表作《美国政治传统及其缔造者》（1948）、《改革的时代》（1955）、《美国生活中的反智主义》（1963）等。霍夫施塔特曾两次得到普利策奖，所著《美国政治传统及其缔造者》至今仍是霍夫施塔特最受欢迎、读者最多的作品。 这本书之所以长期受到欢迎，既要归功于它的哲学，也要归功于它的文学魅力或是它在历史书写上的准确定位。一个现代主义者的决心，那就是把自由主义置放在墨守成规的保守主义与激进的乌托邦主义两极之间。霍夫施塔特在书中体现了二十世纪自由主义一些最优秀的品质，“它戳穿了爱国主义的陈词滥调，却依旧珍视我们国家的传统”。对于美国的政治传统，霍夫施塔特既失望，又尊重它。他对美国生活的多个方面进行了批判，却从未参与过任何反美国主义的流行圈子。 《美国政治传统及其缔造者》由一系列微型传记组成——十个人的人生，从托马斯·杰斐逊到亚伯拉罕·林肯和富兰克林·罗斯福，两组人像（开国先辈，以及“强盗资本家”）——横跨这个国家的历史。如果你读过林达的《近距离看美国》，那么这本书将带你从一个更复杂，也许更加真实的角度来了解美国建国以来的政治传统的形成。让以往标签化的历史印象被解构崩塌，我们可能距离历史“真相”更近了一步，这是一件值得庆幸的事。 这次朝夕讲堂邀请了湖北大学历史文化学院教授张宁老师，通过《美国传统政治及其缔造者》这本书来为大家解读美国的政治传统的行成。来为大家解读美国政治传统的形成，赏析这部曾获普利策奖名著的精妙构思和写作技巧。 「 主讲人简介」Share profile 张宁， 湖北大学历史文化学院副教授， 历史学博士， 研究方向为中国近现代经济史、明清货币金融史。 讲课风趣幽默，深受广大学生一致好评。 「 主持人简介」introduction of the host 余雅君，湖大历史学硕士，非虚构写作、时政观察者。 个人微信公众号：萤火说Yhthinker。 「 活动详情 」Event details 主题｜美国政治传统及其缔造者 时间|6月16日晚上7点~9点 地址｜ 团结大道武车路湖大琴园小区11栋1单元301 「 报名方式 」 Registration form 转发推文朋友圈截图至后台报名 并发送消息“姓名+联系方式” 请提前阅读《美国政治传统及其缔造者》 「 主办介绍 」The host introduced

如果可以，请和我们一起走如果可以，请和我们一起走 李安 玩转湖大 如果可以 请和我们一起走 edite by 玩转湖大 上上个周末，我们发了一条悲伤的消息。 面对各种各样的锤骟，有时言语或文字都嫌轻浮。 其实有时觉得在这里打字，是颇有几分可笑的行为。作为一个非官方公众号，我们失去了曾经赖以为生的一系列便利功能；我们人手不够不能保证追踪你们告诉我们的每一句；我们，甚至不得不很羞愧地讲，能力不逮的时候也有很多。 但是坚持更新的讲座集和灵魂画师呢，我真的发自内心的佩服她们俩。相较于我这种三天打鱼两天晒网型选手，简直应该给她俩一人发一个三八红旗手的大奖章挂上。 （深切鞠躬） 大概就是去年的这个时候，自己坐上了第一次来本部的校车。在朝夕面试的时候，达哥还是DQ问我：“有什么对玩湖的规划？” 隔了这么久，我只记得自己说，要推几个固定的板块。 没想到最后真的跟着两个小仙女也算是这么撑下来一年，慢慢磨出了现在比较固定的几（两）个板块。偶尔上来看看后台的留言，翻翻之前推过的东西，感觉也有一点慢慢冒尖的自豪吧（大雾）。 时间真的是溜起来很快，抓起来很难。玩湖的接力棒在我们手上，也快要握了大半年了。 那天我急着出门，电宝突然失了智。万般无奈之下，我在群里极其狼狈地求助，然后比崔二话不说：“我在桃李园门口等你。”妈呀，五月末那个热浪啊，一定是太热了才逼得我想从眼睛里面流汗的。 那个电宝胖萌胖萌的，安安稳稳护了我一路。 结果没成想，回来我要还她电宝，居然约了三四次才最后交回她手上。 我心头突然涌起一个念头：我们现在开会的间隙越来越久，本来说好的几个活动最后都没有做起来。和达哥聊的时候，他也叹了句：“的确是到了你们该忙的时间了。” 也到了，我们该迎接新一届的你们的时间了。 没错，前面叨逼叨那么多，其实只是为了铺垫一句话： 我们要 招 新 辣！ 欢迎各路有时间有才华的学弟学妹们呀～ 我们需要：1.走心：你得知道这是一个什么公众号，你得对它推送的内容有感觉（吹捧or吐槽都可以），你得希望它越办越好，尤其是在你的手上。 2.走肾：你要有精力有能力来参与这个公众号的运营，它不会每天逼你交作业或者个人总结，但在“搞事情”的时候，不允许你临阵脱逃。 3.针对前两条内容，我们希望你能用一段足够证明自己能力的文字砸过来，或者你是人狠话不多，直接丢来相关成果也可以～不用怕自己不够优秀，证明自己有基本能力，其他一切慢慢来！ 来自她俩的小希望比崔是只鸟：最好有点美术功底…有点子 能搞事什么都好办 雪妮：希望你有想法，有基本的公关和外联能力 来吧快到碗里来～ 趁现在没有四六级考试 趁期末前还有几分喘息之机 快把简历投到碗里来～ 让我们一起走 一起玩  【邮箱是：919981354@qq.com】 【欢迎转发给各院的小鲜肉们哟～】
