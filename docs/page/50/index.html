方小达就查成绩等问题致校长的一封信

一月 23, 2017 发布在 未分类
方小达就查成绩等问题致校长的一封信方小达就查成绩等问题致校长的一封信 原创：方小达 玩转湖大 前言 这或许是最后一次我像这样关心母校的情况了，我知道，坚持很难，难在有太多太多外界因素阻碍。我很开心，曾经也发光发热过，也凭借着自己的努力影响过身边的人，影响过母校。现在，所有的校园微信号都标配查成绩和课表以及图书馆藏书，同学们开始关注加权平均分，知道了绩点的存在，如果有同学因为这些能够更加关注学习而不是学校周边的6个商业街，玩转湖大存在的目的就已经达到了；我也曾经让大家关注过图书馆的事情，并且引发了全校范围内的大讨论，这是我第一次或许是最后一次看到全校同学这样的讨论，很开心，玩转湖大为此而感到自豪。 祝大家新年快乐！若有机会，我愿意用我的酒来换得你的关于玩转湖大的故事。 方小达致校长的一封信 尊敬的校长：您好！ 我是湖北大学2010级商学院的一名校友，校园微信公众号“玩转湖大”的创始人，虽毕业三年有余，但仍心系沙湖畔的母校，因为这里有着我的青春，有着我的回忆，有着我亲爱的老师和同学。我是一个恋旧的人，于母校尤甚，每次放假，我都会特地绕道来母校走走；和同学好友相聚，我也会约着在“同学酒楼”和“小重庆”会面。“日思日睿，笃志笃行”这八字校训，就像家训一般督促着我，让我进步。 从小受到的教育告诉我，要以德报怨，我也时时把这四个字记在心中，我总是告诫自己，生活就是一场修行，我所经历的这一切我接受，我不能要求我的母校只给我带来快乐和幸福，而不带来除此之外的东西。我不能够，看到母校的吉祥物是“借鉴”pixiv上画师的作品就认为母校没有为这个形象花心思（毕竟头饰和配色是改了的），看着学校频繁停水停电就觉得后勤集团做得不好（毕竟宿舍楼有空调了），不能够看着图书馆又开始清理考研物品就觉得图书馆处理问题简单粗暴（毕竟图书馆设备都更新了），更不能够，看到仅仅是因为没有办校园网就不得不在评教的时候奔走于图书馆的同学而觉得母校的信息化建设不好（新版的网站做得比我那个时候好看多了）。 “可是 我真的真的希望 母校能够越来越好 ” 这个学期以来，很多学弟学妹在“玩转湖大”的后台向我们打听学校的教务系统网站为什么不能够进去了。为了这件事，我和小编们还特地开了一个会专门研究这个问题寻找原因，小编发现，通过校园网是可以进入教务系统网站的，但是通过校园网之外的网络均不能。起初我以为是学校网站临时维护关闭了外网的访问能力，我以为会贴出通知之类的说明，于是便进入湖北大学官网和母校信息化中心搜寻，但是，没有找到任何相关说明。同样，我委托玩转湖大的小编以及我在母校的老师打听消息，得到的消息是确实在更新系统，但是什么时候能够恢复外网正常访问，不知道，快好了。这些消息打听工作是我学期初的时候进行的，可是直到现在，教务系统网站依旧无法通过外网访问，依旧没有任何相关通知。 幸运的是，湖北大学官方企业微信和湖北大学官方微信截至目前均能够稳定访问教务系统的成绩和课程信息，同时也有其他的微信公众号通过“技术手段”访问教务系统查询相关信息，但是对于除了查成绩和课表之外的需求，依旧无法满足。 我给校长先生您写信，希望您能够帮帮忙催一下教务系统那边的老师，能不能稍微快一点，尽快让教务系统网站恢复外网访问，让那些没有校园网的学弟学妹们不再因此奔波于图书馆机房，不再因此不能够在家里正常使用教务系统。如果是真的快不了，也希望能够出一个通知，告诉学弟学妹们大概什么时候能够好，让他们不再受到这种本没有必要的麻烦。 听说，学校图书馆的设备又更新了；听说，今年的“三星”比往年的更加精彩；听说，学弟学妹又在全国比赛中获奖了。每当通过QQ空间和微信朋友圈看到这些关于母校的消息的时候，远在西子湖畔的我，仿佛回到了水碧琴园，趴在图书馆的座位上，伴随着沙沙的翻书声小憩；仿佛坐在思睿厅的过道上，聆听大师的教诲；又仿佛踮着脚，在学弟学妹们的尖叫和呐喊声中，透着密密的人群观看“十大歌星”总决赛。每每想到这里，我都会打趣地自言自语道：“呵，好处都让现在的学弟学妹们享受了，我们这些老人都被拍到了沙滩上去了，不过，我的经历你们不一定都会经历，以后遇到你们了，就讲你们不知道的，稳的，不虚，哈哈......” “如果我和母校的故事 能够一直这样 那该多好啊 ”我睡前和吃饭的时候都有刷手机的习惯，湖北大学官方微信更是被我置顶，生怕遗漏任何有关母校的信息，但是，从这学期开始，我就再也收不到官方微信任何消息了。起初，我以为我的手机坏了，可是微信卸载重装N次后依旧如此，作为微信公众号的运营者的直觉告诉我，我可能被拉黑了，在经过多次仔细反复确认之后，我不得不告诉自己的直觉是正确的。我不知道自己做错了什么受到了母校这样的对待，真的不知道，我很伤心。作为湖大记者团新媒体部曾经的技术负责人，我谨慎地寻求答案，觉得这里边一定什么“误会”，希望能够知道原因，我委托在记者团工作的玩转湖大的小编，帮忙问一下，小编告诉我，记者团那边很多人都不知道拉黑这件事，同时她告诉我，能够进入官方微信后台操作的人员不是学生。后来我又委托学校的老师帮我了解情况，同样，老师告诉宣传部的老师也不知道有拉黑这件事。我真的很希望有人能够告诉我原因，哪怕把我批评一顿我都心甘情愿，然后让我改正错误，只要别拉黑我让我看不到母校的消息都行，可是到现在都没有人能够告诉我原因。在写这封信的时候，我做了最后一次尝试，几经周折联系上了湖北大学官方微信的负责人之一“差不多大先生”（为保护隐私用化名代替），很可惜，最后他也告诉我不清楚拉黑的事情。 这件事已经发生了一个学期了，在通过其他途径都无法了解原因之后，我只得写信给校长先生您，希望您能帮帮我，让我这个对于湖北大学还有留恋和不舍之情的校友，知道自己为何被母校抛弃。说真的，伤心之余，我更多的是担心，我担心像我这样待遇的校友会不会有第二个、第三个甚至更多。我一个人的受到这种待遇并不会对母校有什么影响，也就是抱怨一两句，但是这样做让那些和我一样关心挂念母校的校友们怎么办，让那些还在母校生活学习的学弟学妹们怎么办，想到这里，我不免心头一痛，如果大家因为这样的事情而对母校有了不好的印象，进而影响母校的声誉，那就真的只得令人嗟叹了。 最后，感谢校长先生能够耐心倾听一个近乎于话痨的校友的文字，我相信您会是一位为母校学生和校友着想的好校长，希望能够早日收到您的回信，如果有机会，我还是会在去杭州前在母校里走走，点上一碗嘉惠园的热干面，回味当年的校园时光。 祝校长先生您， 身体健康，万事如意 祝母校， 鸡鸣琴园红日上 biubiu报晓传佳音 方文达 微信（825146867） 农历丙申年腊月甘六

跨年这天 你洗了几圈牌

一月 6, 2017 发布在 未分类
跨年这天 你洗了几圈牌跨年这天 你洗了几圈牌 原创：李安 玩转湖大 先申明一句，摘要的句子，版权归作家落落所有。） 跨年狂欢结束了，小长假也结束了，「浪里白条」们也到了该收心备考的关头了。 在翻书之前的最后一次刷微信时（我发四这真的是最后一次刷微信），我不小心向下多滑了两页——除去每日照例嘘寒问暖的父母和有事没事就逼叨叨两句的密友们，聊天列表中出现了很多算不上熟悉的头像。 「谢谢。」 「你也是哟。」 「新年快乐～」 跟在那些头像后面的是我力图避免千篇一律的回复。我绞尽脑汁地运用着我的表情包和标点符号，想要显示出我的「不敷衍」。 个人习惯，近年来渐渐用微信代替了QQ，除去某些班群的通知，已经很少打开那只曾经活跃在桌面右下角的小企鹅图标了。从QQ过渡到微信，好友也筛了一圈儿，多留下了更亲密的、更常瞎侃的人。 但也免不了会加一些并不怎么讲话，或是因为各种各样原因联系减少的人。 「跨年”就像是一场庞大的洗牌，哗啦啦地翻起了后者。 「新的一年顺顺利利万事如意(*^_^*)」 「新年新气象～越来越好～」 等等祝福语纷至沓来，在各种各样的节日零点，手机总会震得不亦乐乎。 我打字打得也不亦乐乎。 同甘共苦捱过高三地狱的同桌给我发：「嘿。最近咋样？」 第一个字跃入眼帘就激我鼻头一酸。微信的数据记录着上次聊天还是去年寒假，一起约着去看老师。我磨蹭，讨厌在冬天早起，和他一分一分拉锯要往后拖时间。他倒是像平时一样好脾气，任着我赖皮，最后看我实在是过分才忍不住：「再不出门，老师们都放假回家了啊。」 我绝不承认是距离带来的慢慢疏远，我们只是平时都有自己的圈子，都在不同的地方忙着不同专业的学习和活动。只是缺少那么一个契机，而「跨年」正是最好的开口钳，恰到好处地撬开了我俩间长久的冰封。 因为活动联系而莫名聊开了的朋友发：「2017新年快乐」 虽然一看这句法措辞，就知道又是群发大法好。不过我依然蛮开心，至少新的一年里，没有被别人删好友，还收到了态度诚恳的一句祝福。我站在江滩跨年的人群里，边笑边颤抖着手指回复：「你也是呀～」 忍不住怀旧的我，总觉得相比高中，大学的神奇性实在是太强。除了同坐同班的关系，还可以是借着花样百出的活动或社团结交到的各式好朋友。不再服从于地域联系而生出的「朋友」，更多地有了兴趣爱好上的共鸣。这样的朋友，淡如水许久，偶尔心念一动也会涌起几分感动的侥幸吧。 但我又不得不承认，这圈浩大的洗牌中，好不容易翻上来的，大多数又会飞快地被新的祝福和信息刷下去。很多人我们就是不再会常常联系了，无论曾经多甜蜜，是否昔日共妖娆，但现在也就真的渐行渐远渐无书了。时间总是很奇怪啊，它一口气能稀释了爱和恨，却无法教我们回到初见时的懵懂和悸动，热恋时的依偎和脸红。 很多人我们就是生不出更亲密的关系，无论心畔多灵犀， 是否爱恨共鸣，但最终也就止于活动中的兴高采烈。有时见着一点相同，便忍不住在心中幻想对方就会是那个百分百男孩/女孩。殊不知管中那一斑，可能也就是豹身上唯一一处美了。 很多人就是走着走着就被我们忘了，说过的刻骨铭心变成了挫骨扬灰。 很多人就是萍水相逢曲终人就散了，相见时把酒言欢不过是逢场作戏。 「哗啦啦」洗牌切牌发牌，我们的每一天都是一场新的赌局。 夜深了，荷官把狼藉收好，身旁同场丢骰子的人也陆陆续续离席。 有人见你出手阔绰，有人见你神机妙算，有人爱你投足风度，也有人就喜欢你眉头紧锁时孩子气的纠结，也有人就看不爽你一举一动；一些从此别过，一些次日还来，还有些将长久相伴。 无论手气好坏，这局你开了，就得努力完满地打下去。 新的一年，话不多说，诸位共进。 跨年后 你的消息列表有什么变化吗？ 反正自各大app都流行起了“年度盘点”之后 小编的朋友圈布满某宝某云的截图了
