投稿 | 你的档案像尘埃【上】

十月 19, 2018 发布在 投稿
投稿 | 你的档案像尘埃【上】投稿 | 你的档案像尘埃【上】 原创：者行孙 玩转湖大 你的档案像尘埃【上】当我收到面试通知的时候，我正窝在沙发里看《神盾局特工》。邮递员把信递给我后转身就走，我保证他连我那句谢谢的毛都没听见。我看了看信，地址栏空白一片，只有三个大大的英文字母——FBI。 话说回来，我不得不佩服FBI的保密工作。笔试过后我到处搜集它的资料，结果除了一大堆以它为主题的小说和电影外，完全没有任何关于它的实质性的信息。 根据信里提供的地址，我在城郊一处偏僻的角落里找到了面试的地方。一眼看过去像是片伊拉克战场上被轰炸过的民房：只有矮矮的两层，深灰色的墙体尴尬地裸露着，墙角的杂草长得肆无忌惮，估计早就在这儿安营扎寨许久了。 我看着眼前的这两层小破楼，再确认一番信里的地址，实在不敢相信眼前的这个建筑物竟然会是FBI的办公地。我半信半疑地走向那一扇长满铁锈的大门，抬手正准备敲，突然一段“十”字形的红光从门里透出来，在我的眼部飞快来回扫动着。我大吃一惊：糟糕！镭射线！ 要没命了！ 而就在我这个念头刚刚冒出来之后，那扇生锈的铁门忽然传来一个悦耳的女声：“welcome。”然后我听见一声清脆的“咔嗒”声，门自动旋开了一个小小的角度。 我长舒一口气：没死！这也不是镭射线！如此看来刚才那段“十”字形的红光应该是类似虹膜识别之类的东西！奇怪，如果那真的是识别器，那么他们又是从哪里得到我的虹膜信息的？ 我推门进去，房间里空空的，四周除了灰尘什么也没有。面前出现了一条长长的走廊，走到尽头看到了一扇电梯门。和外面那个生锈的铁门不一样的是，这扇电梯门的有着漂亮的金属拉丝，像是某种昂贵的高科技设备，散发着高贵又冷酷的气质。我小心翼翼地按下电梯门上的按钮，再次被“十”字形的红色射线笼罩起来。 虽然清楚这大概又是什么识别装置，但还是忍不住让我瞬间停下了肌肉的动作，只剩眼珠子还毫无章法地乱瞟——我总觉得是什么东西监视着我，下一秒就要对我说“休得造次”了。红光退去后，我如履薄冰地准备去按楼层，结果看见它的楼层按钮自动亮了起来。而上面显现的数字却让我脑子一空：-3。 负3楼？这是在和我开玩笑吗？FBI的办公室居然建在地下？ 几秒种后，“-3”按钮上的亮光消失了，电梯门缓缓地打开。就在电梯的正对面，一扇门半掩着，从里面透出来的乳白色的灯光穿过昏暗的走道打在我的鞋上。 就是这儿了！我整理了一下衣领，清了清嗓子，推开门，走了进去。 面试官在桌子的另一边笔直地坐着，他身上的西装合身得像是人体彩绘师画上去的一样，很像让人伸手去摸摸那是否真的是布料钩织的。我注意到在他左胸的位置有一块胸牌，上面只有一个英文字母“G”。同样吸引我注意力的还有桌上放着的那一沓厚厚的A4纸，纸上是些什么？ 面试官看到我进来后，礼貌地起立对我说：“你来了，请坐。” 在我刚刚坐定，还在观察四周环境的时候，面试官G先生（我在心里这样称呼他）说话了：“门口那个虹膜扫描仪是确定你的身份的，关于什么时候收集了你的虹膜信息就不告诉你了；电梯里的红色射线相当于你所理解的安检，当然那要比安检复杂得多；还有，这里只是我们众多办事处中的一个，没有很多事要做，所以只有3层。当然，为了安全，它是在地下。”G先生双手交叉放在桌上，盯着我满是疑惑的脸，“还有什么要问的吗？没有的话我们就正式开始面试好吗？” 搞什么！难道FBI的人真的会读心术吗？！我还一句话没说他就已经熄灭了我所有的疑惑？这不是在拍电影吧？！我本来就一头雾水，被他这么一说我更是七荤八素摸不着头脑了。 G先生盯着我，我端正了一下坐姿，正准备开口，他却恰到好处地抬起他的右手对我做了一个暂停的动作，说：“还有什么问题，等面试结束再问。”话头被打了回来，我略有紧张地做了一个吞咽的动作，乖乖准备开始面试。 G先生：“这次我们只是特地从全国招收档案管理人员，并非专职特工，这点你清楚吧？” 我：“清楚。在你们前段时间的全国范围内突然考试的时候我就已经清楚了。这意味着我面前的这一份工作，是一份文职，而不是像电影里的那样在第一线和敌人展开较量。当然，我个人认为，在档案情报方面的较量，其重要性不输于在第一线展开的较量。” 稍微聊了一会儿后，我没有之前那样紧张了。因为在不久前的笔试中，我排名第一。按理说只需要和G先生进行一段没什么大问题的谈话后，就可以顺利地进入FBI了。我甚至在想，将来的我编号会是多少呢？K吗？又或者是Z？还是最有神秘感的X？ G先生：“嗯，不错。首先我想请你谈谈你对档案的理解。” 我记得在大一的第一节课上，老师就问过我们这个问题。如今我已经博士毕业了，却还要回答这个所有档案工作中都必须面临的最初的、也是最有争议的问题。 我：“我认为档案是将社会活动中直接产生和形成的记录材料，经过挑选过后集中保存起来的记忆工具的集合。” G先生用手挠了一下额头，看得出来他对我这个回答并不是很满意。 G先生：“嗯，很专业的回答。但我希望你在后面的谈话中能够多说说你自己的意见。书我有，字我也认识，不需要你来念。” G先生说的话让我有一些尴尬，专业性回答是我的习惯，经常做研究就变成这样了。我点点头，说：“好，我尽量。” G先生：“好，既然你说了，档案是经过‘挑选’过后集中保存起来的记录材料。那我问你一个问题，没有经过‘挑选’之前的记录材料，你认为它是什么？” 我挺挺腰，接着回答G先生的问题： “没有经过挑选的记录材料，我认为最多只能叫做‘信息’，不能称为档案。因为这里面，还存在着有用信息和无用信息的区别。这里面的有些信息并不存在价值，所以并不能作为档案保存。必须经过挑选后，选出具有价值的信息，才能作为档案进行保存。” G先生：“哦？是吗？有意思。你说信息要经过挑选，有用的信息才能作为档案保存，那么，你靠什么来判断这个信息是有用还是没用呢？” 我：“这在我们专业领域是有一套严密的标准体系来判定的，在这个标准的参照下，我们会鉴别出有价值的信息，作为档案进行保存。” G先生做了一个头部稍微上仰，眉头稍皱的动作，这个动作应该说明他正在思考。 G先生：“这样的话，人的主观意识不会在这里面占很大的成分吗？我的意思是，虽然有个标准，但是如何选择有价值的信息，很大程度上还是取决于人的主观意识吧？” 我真是想不通，为什么作为面试官的G先生，不问我对职业、对未来的规划和打算，却一直在问我这些档案工作中最基本的问题？他到底想干什么？考一场期末试题给我吗？ 但我还是乖乖思索后回答了：“我们在工作中，会尽量不使自己的主观意识影响对信息价值性的判断，我们会严格按照标准来进行选择。”说完后我顿了两秒，忍不住问出了心里的话，“另外，面试官先生，请恕我冒昧，您刚才说不需要我念书，但是您问的这些问题都是书上最基本的问题，我的回答不都是在念书吗？” 谁料G先生好像没听到我说的话一样，坐得笔直的身体往后仰去，靠在椅背上伸了一个懒腰，对我说： “我的意思是，既然在判断信息的价值性的时候会掺杂个人的主观意识，那是不是就意味着可能存在原本有价值的信息，却被当做废品扔进垃圾桶的情况？” “你是否想过，有没有可能开发一个数据库系统，把相关对象的所有信息都录入到数据库中进行保存，然后在需要使用的时候，查找自己需要的信息。目前还没用到的信息，依旧保存在数据库里。让时间来证明，这个信息到底是有价值还是没有价值。想过吗？” 我微微愣怔，但依旧对答如流：“这个问题我没有想过，不过我认为我还没必要这样想，这个假设几乎不可能实现。如此巨大的信息量，里面会有大量的信息重复和冗余，基本上就和记流水账差不多，这样做的话会造成巨大的资源浪费。而且开发一个能够存储如此巨量信息的数据库，以目前的技术水平，我认为是无法做到的。” G先生的脑子到底在想什么？开发出一个巨型的数据库，把所有人的一举一动全都记录进去，这怎么可能？ G先生听完我说的话，笑了一下。他竟然笑了一下，那种笑，我清楚地记得在我们大一的第一节课回答完老师的问题过后，老师也是那样笑的。那个笑分明就是在对我说：我能够理解你的无知。 我感到莫名其妙，G先生为什么要做出这样一个表情？我看见他笑了过后又重新靠近桌子，双手交叉放在桌上。脸又重新变得严肃起来，估计是问答环节就要结束了。 G先生：“好，我再问你几个问题。你认为档案的作用是什么？” 我不假思索：“作为凭证，一切的凭证。为一切生活、生产活动提供凭证。小到个人生活，大到国家生产，甚至整个文明的延续传承，档案都可以发挥它的凭证作用。” G先生：“好，按照你说的，必须是经过‘挑选’过后的、有价值的信息才能称为档案。对于一个国家，一个民族来说，这些记录材料肯定会有专业的人员来进行挑选。很明显，全国各地都存在的档案馆里面就保存着这些经过挑选过后具有价值的信息。那么我问你，如果是一个普通人，一个再也普通不过的普通人，他的档案谁来进行挑选，谁来保存？” 我：“假如是一个普通人的话，这份工作国家会为他做的。从出生到死亡这一段时间内，重要的事国家都会为他进行记录。” G先生：“重要的事？什么是重要的事？出生日期，死亡日期？籍贯信息，家庭住址？又或者是身高体重，民族和信仰？还是处分违规，成绩和学历？如果现在立刻马上，我想要这些记录在档的档案以文字的形式呈现出来，你觉得能有几张纸？” 说着G先生拿起他的右手，放在他面前那沓厚厚的A4纸上拍了两下，用一种质问的目光看着我。他在等待着我的回答。 我咳了咳，这个问题很棘手。如果是一个再也普通不过的普通人，他的记录在档的档案没有经过后期专业地收集和整理，想要立刻马上把他的档案以文字的形式呈现出来的话，内容最多不会超过5张纸。 但是如果我这样回答的话，G先生肯定会接着问：你说档案是个人生活的凭证，那么如果一个人的一生，只有5张纸的记录内容，这5张纸，能够作为一个人一生的凭证吗？ 我迟疑不决，有些犹豫：“可能... ...最多十几张吧。” “5张，最多5张。”G先生伸出5根手指对我说。他又一次猜中了我内心的想法。 我不知道接下来的话该怎么去接，他问倒了我。空气安静了几秒，一丝丝紧张又重新在我们周围蔓延开来。 这时候，G先生开口说话了：“你们这些搞学问的，总是说现在社会大众对档案的重视程度不够。你说，如果档案对于社会大众来说只是档案馆里束之高阁的那数不清的国家和政府文件；如果一个普通人的档案只有可怜的5张纸的内容；如果社会大众从生到死，连档案是什么都不知道也能活得好好的话，他们怎么会对档案产生重视？” “所以啊，我们这次招收档案管理人员，就是为了改变这个现状，改变社会大众对档案的认识，让每个人的档案充分发挥它的作用。我们要做的就是——管理全社会每一个人的档案。哦，按照你的说法的话，就是管理全社会每个人的‘信息’。”G先生并没有抬头看我，而是边说边翻开他面前那一沓厚厚的A4纸，漫无目的地看着。 我的一丝丝不安感瞬间就被无比强大的好奇感所淹没，A4纸上的内容，终于要揭晓了吗？ 在巨大的好奇心的驱使下，我继续和G先生的对话：“以目前的技术，想要对一个普通人生活中的细节进行全方位的记录从而形成详细的档案，是不现实的。所以您说的‘管理全社会每个人的档案’，我认为是不可能实现的。” 我忐忑地等待着G先生的回答，想不到这个时候G先生竟然把话锋一转，问我：“你《神盾局特工》现在应该是看到第三季的第15集了吧，怎么样？剧情很精彩吧？” 什么？！他怎么知道我在看《神盾局特工》？！而且还准确地知道我看到了第三季的第15集？难道是那个邮递员？没道理啊！那个邮递员连门都没进，那个角度怎么可能看到我在看什么？难道他们在监视我？ 突然我眼前一黑，糟了！信息泄露！
