一个大学生是如何被吃掉的

十月 19, 2018 发布在 未分类
一个大学生是如何被吃掉的一个大学生是如何被吃掉的 原创：李安 玩转湖大 定制故事006 真实性：75% 故事人：蒋越 来自主人公的话：我一直想说服自己相信：当你走出那座房子读书立业之后，你就不再是“原生家庭”的附庸。它可能是你心中永远的一颗梅花痣，但不会永远是你脸上的半面青黑瘢。 久未活跃的“闯汉群”突然蹦出消息：“某某找到了，长江里。” 某某是我们那届成绩最好的，再加上定向生的指标，他顺利进入了汉立大学。我们一起坐火车去报道的路上，他教我们几个用了智能手机，还建立了这个“闯汉群”。 起初它总是弹出各种各样的问题，基本都是某某在回答我们；后来一度又成为我们交换红包、表情的小小阵地，再后来随着某某的忙碌，这个群也就越发地沉寂了。 如今，这个群被这样的消息重新炸亮。我看着手机屏幕，百感交集。 报道完，简单铺好被褥之后，我和隔壁宿舍的哥们一起跑到楼顶天台——很不幸，我俩没抢到住一个宿舍的殊荣。他往楼下看着，挺安静地叹了口气：“其实，这里晚上也是黑乎乎的啊。”我看着再远一点的地方，楼上的led灯闪着，又被雾罩着，似乎和我们一样满身燥热。 手机屏幕突然亮起来，某某在群里说：“明天咱们去一次黄鹤楼吧？” “好呀！”“没问题！”“走走走！” 但当某某带着我们在黄鹤楼下站定的时候，人群虽然不断把我们挤得翻来覆去，却没人踏出向前的一步。80块钱挡住了我们瞻仰课本的神圣愿望，几个人静静地看了很久，炙热烤的人背后生汗，犹如虫子般一条一条爬过皮肤。 最终我们一同放弃了登楼的欲望，心照不宣地换到江边去挤轮渡。 我不知道怎么描述2元钱换来的夕照长江，它似乎还是我曾见过的无数次夕阳的旧面貌，但是好像也更加壮阔，混杂着庸庸碌碌的人声和交杂的汗味，大大咧咧铺陈在并不算干净的江面上。水中的亮光似乎饱含轻浮，酝酿着滔滔不绝的热意和油腻——早上吃的热干面，油盐酱们经过了半时的颠簸，和着硬邦邦的面条摩擦着我的胃部。 我吃不惯这样的面，但我和他们说：“好吃。” 某某目光荧荧，他看我的眼神似乎也有半分羡慕。我注意到他吃的时候，也有好几次不得要领地被噎着或是梗住。 我知道某某是在高一的时候，他是全镇的中考状元，比我活生生高了20分，被老师带着去市里玩了一圈。回来的时候整个人神采都不一样了——作为仅次于他的第二名，我一边心服口服地看着他次次在各类考试中碾压我，一边锱铢必较地盯着他的一举一动。他其实不算是人群中十分打眼的类型，个子不高，常常会不自觉咬下唇。 如果刚才我看到的半分羡慕不是幻觉，我大概宁愿从此一天三顿热干面。 某某那天走的十分匆忙，这个“十分匆忙”几乎席卷了我之后对他的所有记忆。他打字很快，话多的时候讲语音也很快，一起回家的时候也是急匆匆如风般跑来，检票过了还要再冲刺两米冲进车厢。 他看起来很忙。其实他的确很忙。像是之前的日子一样，他把自己几乎崩成了一根行将就断的弦。 大一快结束的时候，我们约着一起聚聚。对我来说，大学的每一天都是横冲直撞、打马横刀地重塑着我，我见了吃饭从来只吃一点就扔掉的女孩，我见了三天两头换“战靴”的男生，我也见到了带着低低轰鸣的跑车几乎是擦着我胳膊肘刮过去——那一瞬间我感觉身边的风旋出一阵颤栗，胸腔里似乎是自己压抑不住地吼了一声。但我只能紧紧抓住手里的包子，我怀疑这阵风会把它刮跑。 某某熟练地用桌子边儿开了一瓶啤酒，我还曾暗地揣度今天他会不会是第一次来“撸串儿”，但万万没想到他比我们都要熟捻很多。甚至他喊老板的声音里，我听到的都是汉味的赤辣。他和四年前那个咬住下唇但神采飞扬的校服男孩几乎没有差别，只是下眼脸的沉沉乌青让他更多了几分莫名的颓废。 “压力大啊，再怎么拼命学都考不过别人。” “学生会太黑了，导师太黑了，我家教那家大妈也黑…” “恨他妈那群纨绔子弟，有钱玩，有钱学习，妈的。” 酒比串儿更早见底，我们吃了没多少，但喝了不少。某某似乎是醉了，但依然是我们之中最清醒的那个，他冲我们摆摆手，前去结账。 半夜我们没回寝室，一起在路上七扭八扭走着。那段时间我莫名迷上了《空城》，忍不住吼了一句，似乎也顺势吐掉了半口浊腥的酒气。某某在我旁边，轻轻和着调子哼哼。我锤了他一把：“会唱就唱出来啊。” 某某摇摇头，但另一边响起来稀稀拉拉的几句。 “这城市那么空 这人海风起云涌 能不能再相逢 这快乐都雷同 这悲伤千万种” 某某看着我们在街上乱吼，似乎是张嘴也唱了一点。过了一阵，他提议，一起走去长江大桥看日出。 日出其实没什么值得看的，在老家的时候，我们都在自己上学路上或多或少见过那么几次。走了三公里多之后，胃里的酒意都随着膀胱的几次松紧烟消云散了，但到桥下的时候，大家还是忍不住东倒西歪瘫坐地上。只有某某一个人扶着栏杆强撑着不倒，看样子像是直愣愣望着长江水。隔壁宿舍的哥们大喊他：“嘿，某某！过来！” 他似乎没听见，又仿佛是满眼还沉浸在闪着微微粼光的江水中。 那一喊没震着他，倒是把我弄醒了几分。望着他靠着栏杆的样子，我不禁想起了他亘古不变的QQ签名：“飘飘乎如遗世独立，羽化而登仙。” 我一直觉得普天下的水没什么不同，天下之莫柔弱于水，蜿蜿蜒蜒拖拖沓沓随着石木砖墙哼哼唧唧刺刺啦啦哄哄闹闹地走。挡不住水底的弱肉强食，只能坦白白映些头上已有的镜像。 我的期末比别人都要更紧张，或者说我们几个人的期末都十分紧张。上大学之前，高中老师给我们灌输的不是“上了大学你们就解放了”，而是“上了大学你们就能拿8000块的奖学金了”。我没有别的愿望，我只是逼着自己囫囵地把所有的知识点都生掰硬嚼下去。一遍不行，那就两遍。我相信抄过三遍的东西就不会忘，背过五遍的东西也不会忘。 除了兼职，我几乎把所有的时间都扔进了图书馆。我相信那时候的某某更忙，尤记得那时候忙里偷闲刷了两眼空间动态，还看到他在为学生活动做宣传。我为他浩浩荡荡的几百赞添了一砖， 尽管如此，但我们都没有在来年的春天如愿拿到八千的奖学金。 某某酒后念叨的“纨绔子弟”拿到了，据说是初中就跟着学长混奥赛队，可惜高考失利，只能来汉立大学。平时课日听课少泡实验室多，但逢周末必然呼朋唤友出去畅玩。据说最神的一次是踩着线23点整交完了作业，又出去玩到大半夜。第二天一早继续神采奕奕上课——我想，这应该是货真价实的“神采奕奕”了吧。 我们是不一样的人。 有些人的神采奕奕是天赐的金勺银著，不用太大力气即可得便利铺路，从小时候就不断被捶打磨练接触各类顶级人才，他们的人生是一场又一场的Doodle Jump；有些人的神采奕奕却是吞糠咽菜后的蜜脯，费力攀爬才万分艰苦地抵达别人的起跑线，窥见半分光明稍稍允许自己喘口气的偷闲，他们的人生则是无穷尽的Temple Run，何时停下，便被何时摧毁。 我坦荡荡承认自己是后者，甚至贪婪不顾吃相地抓住一切贫困助学金。某种程度上，我宁愿一遍遍撕开自己贫穷的伤疤，也不希望难得有集体出游的时候，只落下我一个孤零零。 某某不同，他把全部的希望都悬系在奖学金上。学生会和社团疯狂压榨着他的课余时间，再除去背书的时间，他兼职的工资差不多只有我的一半儿。只得拼命节约饭钱，以备不时“活动”之需。我曾想问他何必把自己逼到如此境地，但当我见到他气质淡雅的女朋友之后便无此疑问。 如果我也有这样的一个女朋友，我一定不会想听到别人说“搞不懂为什么她找了那个拿助学金的穷小子”。 一方面，她的出现极大极大地安抚了某某紧绷的神经和时刻争攀的极性心态；但一旦离开她，他就必须面对自己依旧困窘的现实；尤其是在每次和她约会之前，他都必须要更加努力地勒紧裤带那么一两天，单靠灌水欺骗自己空荡荡的胃。但挽起手的那一刻他又觉得一切都无比值当。他和我们说，多亏她出现，多亏她降临，否则真不知道自己如何才能压制住心里那头无差别无间断狂吠的疯狗。 寒假快结束买车票时，我听他同村的小哥讲，他这个年过的不太愉快。 家里本就只有父亲一个劳动力，母亲在他小时候就落了残疾；但去年秋末，唯一的劳力也由于过劳倒下了。父母的焦虑统一爆发在他未能拿出好成绩的那一刻，再穷的僻壤都燃起来爆竹声声，唯有他家半个正月都过得惨惨淡淡。 报道的车上，我们问某某家里怎样了，他朝我们半颓着叹气：“嗨，现在看看读书真是没用，还不如跟我表叔搬砖去。” 几个人一霎无话。 我们都或多或少有这样的焦躁和无奈，这种情绪不属于自卑，和痛楚也不沾边，我们只是焦虑。锦衣玉食的“纨绔子弟”可以优哉游哉，但我们需要卡着点去一个又一个兼职或是家教的地方，买东西要拼命货比三家一不小心就因为贪便宜被假书贩子骗到；也不能倦了就去网吧开黑放松，渴了就随手买杯十几元的奶茶，没法毕业几年就拿了爸妈的钱付个车房首付。我累了就睡，热了洗凉水澡，饿了吃个饼子喝口水就算。 贫穷给了我们最深重的不安全感和惧怕。 学校不再是幼时梦中的象牙塔，它是进入社会之前的指压板，扎了人无伤无血的痛。从乡村到城市，从镇级中学到重点大学，平台上归拢了越来越多的人。聪明的人太多，努力的人太多，各项全能的人太多，从起跑线就超过我们的人不计其数。我们曾经最赖以为生的成绩之刃，早已被劈砍得破破烂烂，毫无锋芒。 为了有朝一日能喝同样的一杯咖啡，为了渐渐抹平父辈祖辈那里开始的鸿沟，我们只能不断地催逼自己向前，向前，更快更快地向前。
