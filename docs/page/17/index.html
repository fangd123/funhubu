推迟了好多天，这下《变·迁》终于来了

十月 26, 2018 发布在 未分类
推迟了好多天，这下《变·迁》终于来了推迟了好多天，这下《变·迁》终于来了 玩转湖大 纪录片《变·迁》是由湖北大学新闻传播学院的学生团队拍摄制作的，并由该院老师们的多方面指导，现已完成前期拍摄，并推出预告片。 问题一：《变·迁》主要纪录了什么内容呢？该片主要纪录了在汉正街整体升级改造的大背景下，汉正街已经消失、正在消失或者即将消失的一些看起来很普通的生活元素。该片以一天的时间为主线，通过三位主人公的日常生活，力求展现出一幅汉正街式的清明上河图。 问题二：为什么会选择拍摄汉正街的变化？ 首先来说汉正街，汉正街是原本是一条著名的商业街，迄今已经有500多年的历史，如今的汉正街泛指由武胜路、沿河大道、民族路、长堤街围绕的以小商品批发而闻名全国的商业区。由于经济的发展，汉正街的经济地位在逐渐下降，加之街道建筑混乱，极易发生火灾等原因，武汉市政府决定对汉正街进行改造并建设武汉汉正街中央服务区，而小商品批发这一功能则逐渐整合，整体转移至汉口北，使小商品批发继续更好的生存下去，也使汉正街更具新的经济活力。 其实一提到汉正街，人们往往会想到汉正街的经济地位，而纪录片《变·迁》则反其道而行之，从汉正街的文化角度来着手。但不得不承认的是，汉正街的文化，也恰恰是在汉正街这种极具代表性的经济基础上形成的。有一句话叫做“汉口本来无土著，九分商贾一分民，”因此该片所讲述的汉正街的文化就是汉正街人的固有的生活仪式，或者说生活秩序以及一些汉正街的文物遗址、历史传说。而这种生活秩序里不开汉正街人，这里所指的汉正街人包括汉正街的居民、商家甚至是扁担。但是如果汉正街整体升级改造之后，这些人就会到不同的地方开始新的生活，那么这就会导致一种文化的断裂，因为文化的传承，离不开人。 当然上面所说的这种现象几乎会出现在全国每一个旧城改造的地方，但是毕竟汉正街有它的历史和知名度，也有较高的关注度，这也是选取汉正街为拍摄对象的重要原因之一。 问题三：为什么要拍这部纪录片？ 导演王玮琪：“拍这部片子主要有三点原因，第一个原因是来自老师，学院的聂远征老师主要研究代际传播，他早在14年暑假前就曾经跟我提到过这个想法。之后因为种种原因，在15年一月才开始正式策划，五月开拍。当然其中江龙军老师也说他年轻的时候也想拍一部纪录片，但因为种种原因一直没能拍成，他也非常希望自己的学生能拍一些东西出来。第二个原因是因为我的父亲在刚送我到武汉的时候，就问我的室友汉正街在哪儿，我当时就不明白汉正街是什么？所以说汉正街对于我们上一辈和我们这一代人，影响力是相差很多的，这也算是一个汉正街情结。第三个原因是因为我自身是一个比较怀旧的人，试想在若干年以后，当我的孩子问我“爸爸，你在武汉读过大学，汉正街很有名，是一段历史的见证，那当时的汉正街是什么样的呢？”我如果不能拿出一个东西看给他的话，我觉着会很遗憾。” 问题四：如何评价这部纪录片？ 相对来说《变·迁》的拍摄周期已经算很短了，这就意味着很难从时间的跨度上打动观众。个人评价这部片子画质很一般，也有一些不够成熟的观点，故事性也不是特别强，但这部片子是一部既抓细节又观全局的片子，会给观众呈现出一个非常朴实的汉正街，其中包括汉正街老人面对拆迁内心的真实想法，以及扁担一家的生活，有关汉正街的典故等等，都是很值得一看的内容。 问题五：《变·迁》的正片会在什么时候上线？ 预告片出来之后，大家可能会有疑问，正片什么时候出来呢？其实确定的时间现在也没有定，已经来临暑假是团队成员事情比较多的时候，实习，复习雅思等等，但预计成片会是在八月下旬与大家见面。 问题六：制作团队的阵容是什么样的？ 纪录片《变·迁》的主创团队成员主要有七个学生，其中六人来自湖北大学新闻传播学院2012级播音与主持艺术专业，一人来自2013级广播电视专业，在片子策划和拍摄的过程中，大家都为这部片子付出了很多，后期剪辑大家依然会一起协作，齐心完成。 强大的团队阵容来了： 王玮琪：导演、摄像，负责摄制组总体统筹和拍摄工作； 林云舒：总策划，负责纪录片拍摄制作全程策划和前期调查工作； 汪嘉欣：策划、宣传，负责协助总策划完成全程策划调查和宣传工作； 张 宇：策划，负责协助总策划完成全程策划调查工作； 王志强：监制，负责纪录片的拍摄和剪辑工作； 萧济康：监制，负责纪录片的拍摄和剪辑工作； 郑晨曦：摄像、导演助理，负责协助导演完成总体统筹和摄像工作。 除上述七位摄制团队核心成员以外，还有如下七位老师作为纪录片《变迁》的顾问和特别顾问： 聂远征：湖北大学新闻传播学院副教授； 江龙军：湖北大学新闻传播学院副教授； 路俊卫：湖北大学新闻传播学院讲师； 陈天依：湖北大学新闻传播学院讲师； 杨璐旻：湖北大学新闻传播学院实验师； 特别顾问： 王仁昌：作家，汉正街文化研究学者，汉正街乃至全国第一批103家个体户之一，著有《风流巨贾》、《汉商》等长篇小说，其小说《风流巨贾》被改编为八集电视连续剧《汉正街》。 当然，除了主创团队以外还有很多人的帮助，比如新传院学工办的老师们在各个方面的大力支持，武汉朝夕青年空间偿提供剪辑设备和场地，志愿者们的无私奉献，外院大神的精心翻译，还有身边同学朋友乃至家长们的理解和鼓励，啦啦啦~~当然还有玩转湖大的平台支持。这些都是这部纪录片可以顺利拍摄完成的原因。当然至于后期成片的剪辑，依旧离不开大家的帮助，希望大家可以继续关注纪录片《变·迁》的动态，关注我们身边正在消失的看起来很普通的生活元素。
