玩湖@你——这一次我们来真的

十月 21, 2018 发布在 未分类
玩湖@你——这一次我们来真的玩湖@你——这一次我们来真的 玩转湖大 >>>>给湖大学子的一封信 春来了。 图书馆前的一棵树一夜之间开了花，路过的人都要驻足观赏那么一会，或许是冬天太漫长，亦或是从来都是脚步匆匆略过，我们都习惯了仰首是春，俯首既是秋。 开学已然一周，假日里的舒适日子也到了头，大家都又重新适应了校园生活，而玩转湖大仍然沉寂着，唯有查成绩的时候他才会出现在大家的视线里。其实玩湖早已踌躇满志，想给大家搞个“大新闻”。 玩转湖大经过一年多的发展，到目前已经是第三代团队，有前人打下的良好基础，有今人不懈的努力，我们已经站稳了脚跟，于是我们就在想，在这平静中我们该如何突围？ 在运营过程中，我们不断发现湖大学子给予我们的惊喜，感叹于此地藏龙卧虎，我辈中亦有众多人杰。经过几番周密的规划，我们决定开设一个全新的版块。 这个版块的核心便是原创内容，你若是妙笔生花的才子佳人，可以写写书评、影评、散文或者诗歌；你若是爱好摄影绘画，照片、手绘、微电影也同样精彩；你若是有自己的歌曲创作，原创歌曲也是别有一番风味…… 任何形式的原创内容，玩转湖大都欢迎你的投稿，投稿邮箱：funhubu@139.com，统一格式为【类目】+标题，例如：【书评】一个人的轮回——《生死疲劳》。 玩转湖大会精挑细选出最优质的作品，定期分享给大家。并且，我们也会尊重并保护作者的版权，会为你点亮原创标志并为你维权。同时，我们与湖北大学青年传媒中心合作共享优质资源，你的文章将有机会发表在青春飞扬网站、《沙湖青年》杂志及微信平台上。 朝看水东留，暮看日西沉。玩转湖大自成立以来，宗旨就是公益性、服务性，我们的成员也是一群志趣相投、热爱生活的小伙伴，一直以来都毫无怨言的为着这个平台付出着，但是自这学期开始，大家的学业负担逐渐加重，招新的工作也提上了日程。任何有计算机编程、文字编辑、文章撰写以及公关与管理能力或者学习兴趣的15级同学，都可以加我微信：Dangeremain，我们会安排后续的事项。 入春阳光静好，岁月平静，玩转湖大未来的路还很长。将后来，我们还有一些开设新版块的设想，比如“图说湖大”，或许在不久后就会和大家相见。 最后，很感谢你在大学生涯中使用了玩转湖大，或许自从我们遇见，时间就将这么静悄悄的过去四年，你进入湖大，我在这里，你离开湖大，我在这里，离开后，你记或不记得我，我都在这里，不来不去。 玩转湖大成员代表：邓桥 2016.03.02 投稿邮箱：funhubu@139.com联系人微信：Dangeremain
