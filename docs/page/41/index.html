所有人都知道职涯规划的重要，但是谈到人生伴侣都在讲缘分。不觉得很可怕吗？明明配偶就是生涯规划最重要的另一半，这个人在法律上可以跟你分财产，拔插管。多数人却用一种碰运气的方式看待恋爱与婚姻。等待缘分，等待心动，等人来追。 用摸瞎靠缘分的方式来找下半辈子的人生伙伴……不觉得很可怕吗？人们普遍认为”太有规划的追爱不算真爱”。鄙视有规划过的寻爱之途，认为不够随机的恋爱都隐含着低劣盘算。
