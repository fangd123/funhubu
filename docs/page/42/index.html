人生啊，欢愉有限

十月 18, 2018 发布在 未分类
人生啊，欢愉有限人生啊，欢愉有限 原创：李安 玩转湖大 认识的几个朋友，在上了大学之后都吐露出受抑郁症困扰的事情，有人是焦虑转抑郁，也有人是双向而偏抑郁。诱因不详，大概每个人总有那么一条压碎自己的稻草，模样千变万化。 高考以后才渐渐明白，人生最崎岖的道路不是五三王后雄，而是全然无知时需要的探索。前一步海阔天空，但也很可能粉身碎骨。错一道题，尚且有答案可以改正弥补；走错一步路，却是迷迷茫茫不晓得何日才得报应。我们每迈一步都变得瞻前顾后，恨不得先找一参考标准照着走。 大一就去某司实习的发小。在我眼里早已是不折不扣的“人生赢家”，还是半夜发了一张心理医生警告邮件的截图。删除之前，他自己对着那张图片絮絮叨叨：我想和爸妈证明我一个人这么小出来也是没问题的，想和曾经花掉的钱说我这么快就要把你们挣回来了。我狠命砸碎了之前所有的玻璃碗，为的就是证明我本是一块儿熠熠发光的星星。 素来熬夜刷微博的我，仿佛灵魂都被他这段话抽去了未眠的混沌。 另一位“某v”，乃是我八竿子打着一裙角的同学的学长前女友的闺蜜，七绕八绕留了份关注。平日里看她嬉笑怒骂，信手拈来西欧美术史艺术史什么的，常年活得没有烟火气。就连偶尔的素颜自拍，都精致的宛如影楼p出来的人像照。一段时间长久不见她发动态，突然讲话，却是长长一篇文章。 “爸妈之前给我玩笑，对我念书的投资就像是养小三，每年花钱可以如流水，但最终一定要回家给我们看看依旧身康体健、心智多上两层楼。开始我也是当笑话听，渐渐觉得自己务虚也甚得轻松快活。可惜只有陡生变故之后，才发现自己平时摆弄的绣花架子才是最大的笑话。” 无意揣测他人，只是有幸走过一样的道路，看到这些难免颇有几分触动。奈何自己没有他们般勇于启口。毕竟抑郁的人，大抵怕的比愁的都要多：怕自己讲的遭人笑话，也怕说出口空落落无得回音，更怕别人过度解读，背里暗暗戳上脊梁骨——“嘿，就那人，千万别招他，抑郁可是不得了。” 从人嘴里飞出来的灰色叶子，落在我们身上，总要刺得更痛些。 说来不齿，人家的抑郁总能放到正儿八经的理由上。我的画地为牢，却是缘起一团自己也讲不清的乱麻。说是该着高考失利，却也可能是怪自己先前遇人不淑，还可能是压抑心理的突然爆发。重重的砖石压下来，我就落进这盘倒霉的塔，怔怔不知何处去了。 讲话讲不好，记事记不住，抑郁带来的记忆里下降和行为失调让人一瞬间退化成无能的原始状态。每天渴望有个蛋壳能包着自己，不碰外界的风风雨雨。一点儿小小波澜，到我这儿就是一场未遂的海啸。但心里还有个声音一气儿着碎碎念：你不能倒啊，你不能抑郁啊，你开心着呢，世界美好着呢——该着我这个“顽固不就医派”继续受折磨——讳疾忌医把自己抻到了更病态的极端。 所以在此告诫各位，常去心理咨询，尤其是自己感觉不良好的时候。比自己硬撑着好一万倍。 那时候，我脑子里只有三个念头：和人相处时候是“啊呀我又做错了，还是乖乖闭嘴自己玩儿蛋去吧”；大小考试前是“这么简单的东西怎么就是记不住，你真是个超级大废柴啊”；剩余时间都是“世界这么好，我却这么糟”。每天简直像固定程序的机器人，脑袋里都不会跑别的数字了。抑郁带来的迟钝和记忆力衰退不断加重着自己的焦虑，一个完美的死循环就这样套上了自己的脖子。 幸运的是，我没想过死，但我也亦觉人生无可恋。 恢复的过程让人难以预料。一朋友拉我经常出门浪，吃饭唱歌跑步看电影，只要不用到我大脑和细微神经，也远离比较的排名设置，我的快乐似乎就能颤颤巍巍回到脑壳儿里。筋疲力竭的快乐，饱食至撑的快乐，纯肉体的低级欢愉竟是治愈抑郁的良药？ 我羞耻到不敢承认。 但不得不讲，事实如此，希望你难过时也能试试。 一次豪饮，一场狂笑，胜过整整一盒的氟西汀。我们和父辈们不同，不敢无忧虑无旁骛地埋头苦读，象牙塔之外没有什么能承接我们印刻铅字的灵魂；我们也和弟弟妹妹们不同，不敢肆无忌惮地哭闹，明白社会的冷在于无人在乎你的柔弱。我们活在这个年纪和这个时代，早先背上了不由自主的重担。熙熙攘攘的人流推着我们，除了三和大神，无路可退。 人生啊，欢愉有限，青春无限。学会轻轻放一放，歇一歇，跳进莺歌燕舞从中赏赏景听听歌，才是青年之要务。把握及时行乐和不忘初心，怕是我们接下来的人生天平，常常需担着的项目。 新学年，新学期，玩湖也没太多说的。 就祝每一位，都能开心自在，平安顺遂。

不，我只怀念逝去的弹幕不，我只怀念逝去的弹幕 原创：春宵苦短一起农药 玩转湖大 知道b站（bilibili）还是在初中，那时的b站是二次元同桌的心头好。 她神采飞扬地边看边讲：“昨天我还帮xx答了一百道题，她一会请我吃饭。” 我好奇地问：“这是什么网站啊，答什么题啊？” “就是一个视频网站。答了题才能进去，有问你各种关于动漫的小问题，没看过的都答不上来。” “噢。”我不明觉厉地点点头。 毕竟那时候，掳获我少女芳心的还是“快车”，想看什么下什么。摆脱拨号上网的龟速模式后，回家先开电脑下个“资源”，睡前把电脑屏幕调到最暗，为了不碰响椅子跪在瓷砖上偷偷看完那部热乎的电影——不得不说，除了国产院线片，剩下的40%阅片量，都要归功于那个黄色的小图标。 剩下的60%，则是迅雷和ab站的功劳。它们在我蔓蔓的青春里渐次登场，填补着无数个脸上戴满显示屏荧光的岁月。如果认真地统计一下（（=去掉农药和某宝））我的时间消耗，大概是由某云、某站笑傲榜首的吧。 一个统治我的耳朵，一个统治我的眼睛。 在我一个人对着电脑哭哭笑笑的日子里，我知道还有遥不可及的陌生人和我一样。我是有归属感的，我不是茫茫群生中被格格不入的那一个。 这两天，我一直在思索要在假期写些什么推给你们看。要写写湖大现在有什么趁着假期作出的改变之类的吗？比如大家心心念念的空调热水什么的。 但是很尴尬，我一打开我的QQ，就看到住同栋楼的小姐姐们控诉宿管阿姨“一言不合就断电”的行为弄坏了自己的网、甚至只是在中午休息片刻的机会都要被热醒（注：宿管阿姨看到门口有餐盒垃圾会给寝室断电，但有时候只是同学们刚吃完没多久，还没来得及去扔便遭到了“鲁莽断电”）。 因为修地铁，所以说好的空调热水还要再遥遥无期地等下去。 因为保持楼道整洁，所以垃圾出现了就要直接掐电予以惩戒。 那我们只谈风月。那我们推荐一些经典的影视作品给你们看好啦，看剧配西瓜才是夏天的真正奥义。 正在辛辛苦苦整理收藏夹，昨晚一登陆，差点以为自己进了假的b站： （随手截了两张图，大家感受一下。） 一天过去，B站官方给出了回应 的确，B站那么多免费神剧，大多是由UP主私人上传的，存在却不合理，早就该予以整改。但对于那些上世纪50年代的外国影视剧呢？在《尼泊尔公约》保护之外的影视剧呢？ 版权意识的逐步加强是件可喜之事，但大抄特抄的《三生三世十里桃花》呢？《夏至未至》呢？国产保护光环的抄袭就不算侵权了吗？ 是要我们都去看手撕鬼子，或者抄袭韩国的综艺吗？ 怕是“未审先播”才是真正的掣肘。没有相关资质，没有许可证，你的精神花园早就应该关门大吉。 对于有些事，我们的愤怒和口不择言，并非昏了头一意给这件事猛泼脏水。只是罚也好，惩也罢，换一种实现方式似乎也未尝不可？ 考了59分回家，是面对父母劈头盖脸的打骂会让我们下次考的更好，还是共同攻克这张卷子上出现的纰漏会让我们下次考的更好呢？ 为何不能先用其他方式以示警戒，一定要在炎炎的武汉、毫无空调的寝室里直接断电逼醒睡在高层的同学呢？他们可能只是刚吃完、想着小憩一会醒来就下楼扔掉的啊。 为何不能先颁布内容规范，给予整改办法，给出一个时间期限让平台内的生态系统完成“自净”的环节，而是一定要利用如此秋风扫落叶的方式呢。辛辛苦苦下载、转码、压缩、上传的一切瞬间就消失不见，曾经承载我们最多的感情宣泄的弹幕们更是无从恢复。 我们不反对惩罚，我们只是害怕。 不想睡着睡着被热醒，不想追了一半的剧凭空消失，不想我们有一天失去所有的抗争权利，不想有天我们的后代再也没见过丰美的草原，而发出诘问：“哪有什么‘风吹早低见牛羊’？” 比我大五岁的学长说他们上学的时候，还是从Tumblr上找东西做ppt，gmail用的风生水起。而仅仅距离一个代沟的时间，我们这一代，大概有不少人，连Tumblr和gmail为何物都不甚清楚吧。我怕将来的我，会忘记早先尝过的甜美滋味。 VPN被封的时候，我想起知乎上一则答案，大意是“墙”的存在，是为了挡住心智不成熟者，擅用技术、有一定姿势水平的人还是可以自由来去，这其实是非常绝妙的互联网分级制度啊。 但是当green倒下的时候，我不禁问自己，当初我为那篇回答点下的“赞同”，是不是压死它们的石子之一？ 最可怕的是，当看到自己敬爱的人明明做错了事，我们却还要出于信仰为其寻找开脱的理由。 在互联网地球村的时代，一骑绝尘发展着的我们，却要在国与国之间重新架起藩篱。我们只能吐槽某搜索广告多，某百科不靠谱，某网站资源差，某博客真相少，只能眼睁睁看着人血从黑医院中溅出来，只能眼睁睁看着自己的资料库越来越贫瘠。 维稳可期，愚民不可。保护性防御可期，闭关锁国不可。 不要把仅有的程序错误，变成真正的事实错误。一颗彩蛋 对于已下架的番剧，搜索“哔哩哔哩唧唧”有惊喜哟
