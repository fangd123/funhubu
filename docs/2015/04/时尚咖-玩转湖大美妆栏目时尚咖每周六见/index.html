刚开始学化妆，你应该优先入手哪些东东？

刚开始学化妆的妹子都会有这样一个疑问：“我想学化妆，要买些什么呢？”


大部分美妞都徘徊在化妆门口，不知道该如何入门？看着琳琅满目的化妆品无从下手？还是听着导购天花乱坠的介绍买了一堆自己暂时用不到的化妆品，最后画出一个大花脸？


别着急，大美妞们~小编搜罗了各大美妆栏目，总结出了新手化妆必备的六款基础彩妆产品！有了这6样东东，你就能画出一个自然精致的日常美妆咯~
化妆新手必备6件：
粉底液、蜜粉、眉粉/笔、睫毛膏、眼影、唇膏/唇彩
这些都是最最基本的哦~下面让小编来一一为美妞们介绍：
新手必备底妆类
粉底液/BB霜：
对于入门级的妹子，建议选择液状的粉底，就是常说的粉底液，因为粉底液对于化妆技术的要求不高，只要选对适合自己的色号，然后用手指推匀使服帖即可，呈现滴效果会灰常自然。
PS：建议选色要偏白一号，因为有的粉底会暗沉，尤其是油性皮肤的妹子，氧化现象会更加严重。
如果你自认为肤质状况还ok，那就更easy了，一只BB霜拯救世界，提亮修颜一步搞定！
小编推荐：

美宝莲精纯矿物BB霜

MISSHA谜尚M幻彩BB霜红BB

莎莎价89

散粉/蜜粉：

学化妆的MM或许还不能掌握让底妆服帖的小技巧，那么就可以让蜜粉来帮忙，粉末状的质地可以帮助吸附一些油脂，让面部不那么油光可见，防止脱妆，而且皮肤会看起来很通透哦~
建议选择透明的蜜粉，这样即使你掌握不好用量，用多了也不会被发现~嘘。
小编推荐：

Innisfree悦诗风吟矿物薄荷控油散粉

新手必备眼妆类
眉粉/笔：
眉毛对于整张脸是很重要的！！！（小编尤其有感触）
建议刚开始学化妆的MM用眉粉，眉粉易上色且自然，但是，如果你的眉毛淡到不行，那么就需要使用眉笔来辅助画出线条啦！
颜色的选择上，不要根据你的眉毛颜色，而是要和你的头发颜色相近。
小编头发是深咖色，所以用的眉笔也是咖色的~~
小编推荐：

KATE眉粉

爱丽小屋双头眉笔

眼影：
初学化妆，眼线掌握或许对我们来说还太难，那么，就用最经典不会出错的大地色眼影来打造一个有神精致的眼妆吧~
新手建议入手最简单的粉状质地大地色眼影，用较浅色刷满整个眼窝，用较深色填满眼褶处，眼尾再加重一次即可。
小编推荐：

Za大地色系四色眼影

睫毛膏：
很多人的化妆就是从一只睫毛膏、一支口红入门的~
新手的话，买一只防水的黑色睫毛膏就可以啦！
至于款式选择：如果你的睫毛长度尚可，喜欢自然妆容，那就买纤长效果的；若果你的睫毛有点塌，那就选择卷翘效果，如果嫌自己的睫毛太稀疏，就选择浓密效果的。
当然，使用前别忘记用睫毛夹夹翘睫毛，效果事半功倍！
新手必备唇妆类
唇彩/唇膏：
Ending time.初学者建议挑选最百搭的粉色系或者，玫红色系，简单的一抹唇色，瞬间提亮整个妆容，出门好气色妥妥的。
唇膏显色度更好，上色更持久；唇彩呢会让嘴唇看起来粉嘟嘟，美妞们，依据自己偏好自行挑选哦~
小编推荐：

卡姿兰丝缎美型唇膏（口红）

莎莎价45元
