留学项目逐个击破ing~
1.美国北亚利桑那大学2015年2+2留学项目


凡我校物理与电子科学学院、资源环境学院、商学院、新闻传播学院相关专业全日制本科二年级（2013年入学）学生，均可报名申请；

报名截止日期为：2015年4月10日17:00。


2.西班牙拉曼鲁尔大学2015年本科生交换项目


正式在册的2013、2014级全日制本科生。

报名截止日期：2015年3月20日（周五）17:00。


3.澳大利亚迪肯大学2015年留学通知（会计专业“2+2”本科双学位项目）


从商学院会计专业本科二年级学生中选拔。

报名截止日期为：2015年4月10日17:00。


4.美国加州大学河滨分校（UCR）2015年留学通知 （商科项目）


我校各专业三年级本科生可以报名。经济、管理类专业的学生可选择3+2或3+3项目；其它专业的学生可选择3+3项目。

报名截止日期为：2015年4月1日17:00。


5.美国韦恩州立大学2015年留学通知 （“3+2”本硕连读项目）


从在校三年级（2012年入学）学生当中选拔 。
报名截止日期为：2015年3月30日17:00。


6.芬兰瓦萨应用科技大学2015年留学通知


从在校二年级（2013年入学）学生当中选拔。各学院专业要求如下：
数学与统计学学院：信息与计算科学。
计算机与信息工程学院：电子信息工程、计算机科学与技术、软件工程、通信工程、信息安全。
物理与电子科学学院：电子科学与技术、微电子科学与工程。
商学院：国际经济与贸易、金融、经济学、工商管理、市场营销、会计。
报名截止日期为：2015年3月30日17:00。


7.日本早稻田大学2015年留学通知


拟从日语专业或辅修日语的其他专业二、三年级全日制本科学生中选拔。
报名截止日期为：2015年3月20日17:00（周五）。


8.日本上智大学2015年秋季招生简章


拟从日语专业或辅修日语的其他专业二、三年级全日制本科学生中选拔。
报名截止日期为：2015年3月20日17:00（周五）。


9.关于选拔台湾云林科技大学2015年秋季交换生的通知


免学费交换生2名，自费交换生名额不限。
报名截止日期：2015年4月10日（周三）17:00。


10.法国图鲁兹第二大学2015年留学通知


从外国语学院法语系大三学生为主进行选拔。
报名截止日期为：2015年3月20日17：00。


11.美国加州富乐敦州立大学2015年留学通知
我校二、三年级相关专业全日制本科学生均可申请。
报名截止日期为：2015年3月28日17:00
