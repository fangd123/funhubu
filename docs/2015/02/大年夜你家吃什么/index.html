丸子 玩转湖大
羊年到了~~~

新年好呀，新年好呀，祝贺大家新年好，我们唱歌，我们跳舞，祝贺大家新年好！


过去的这个马年，大老虎被打了，呼格案被破了，郭美美被捕了，北京APEC天蓝了，首届互联网大会召开了，马航遗憾失联了，国际油价大跳水了，连股市都疯了……
湖大小鲜肉赶走老鲜肉，一茬又一茬的新人涌入，一波又一波的老人离去~这个马年，西街没了，二期后门火了，游泳馆没了，实验室有了，湖大学生稚嫩没了，成熟有了……不管什么有了，什么没了，咱湖大依旧是叼叼的内环学校！
这个马年，马上有对象、马上有钱、马上有……whatever，都已经过去了~
羊年，玩转湖大依旧陪着大家，依旧需要大家的支持，依旧努力为大家服务着。

晒出你家的年夜饭
参与说明
都说南北差异大，那小编觉得南北的年夜饭差异也很大，饭的品种，吃年夜饭的时间也都大不同。
今晚就是除夕了，晒出你家的年夜饭，比一比谁家的年夜饭漂亮好看又好吃，看一看你家的年夜饭有啥子习俗哟~
参与方式一：
点击下方“阅读原文”进入微社区晒出年夜饭，并说明家乡年夜饭的习俗。

把你家的年夜饭+年夜饭习俗发给玩转湖大后台，
小编会随时关注后台动态，选出最有特色的年夜饭，后期有惊喜相送哦~~
晒一晒，玩转湖大与你一起过大年。
