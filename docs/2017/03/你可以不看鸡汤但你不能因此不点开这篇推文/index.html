你可以不看鸡汤，但你不能因此不点开这篇推文

原创： 李安 玩转湖大

（本文可能有鸡精味儿，不适者请直接滑到最后）

离开高中一年半，终于敢拉下脸在寒假的时候回去聚会了。其实本也没什么值得「豁出去」的，只不过是高考前牛逼吹得太过，考完了成绩出了，打脸打得太疼，五指印红通通着横竖没法见人。

静言思之，当时那颗「中二」丸还真是让自己神魂颠倒好久。本来就是自己不够努力，大家认真学习的时候自己一心向玩，这驽钝资质怎么可能潇洒考好。一套练习册，每本翻了几页便再无痕迹，买书付钱时的「洗心革面」已早被我丢到千里之外；一本单词书，只有第一章被我细心翻过，按「艾宾浩斯遗忘曲线」定好的计划再次付诸东流；一堂物理课，前五分钟还跟着老师的粉笔字，牛顿焦耳的名字刚念完心就不知飘到哪里去了……稍微认真复盘一下自己的高中生活，似乎来湖大，都算是「侥幸」了。

那还有什么丢脸的。

技不如人，还不努力，坦荡荡面对自己吧。

到了早定好的位子，我先看到了她。她把头发剪短了，底下烫了小内扣，远远看去皮肤白到发亮——和我打招呼的时候笑出一口白牙，几乎让我自惭形秽了。

我俩倒也算某种程度上的同病相怜，双双栽倒在最擅长的语文上，一起分享过的曼妙梦想摔碎在普通一本里。席间俩人心照不宣，没有人提起那倒霉的两个半小时。一边就着重庆火锅的鲜香红油，她一小口一小口地咬着涮好的小酥肉——我几乎没法把面前这个精致的形象，和当年陪我大口吸溜路边凉面的马尾辫女孩叠到一起去。时间过的真快啊，一刀刀快得近乎无影。

「我们书法社社长特别厉害，你看这是他写的《蜀素帖》。」她又吃下一块嫩牛肉，唇釉还一丝不苟着，调出手机相册指给我看，「开学招新那会，我们直接让他当场表演，一堆小学妹全围在那！」

她继续给我展示着学长的字，握着手机的手指纤长如白葱，绘了小圣诞树的美甲几乎吸去了我的所有目光。「啊不不不，这是我写的烂字，别看别看。」她一个手快，滑过了头，又赶快用手挡住，娇赧地朝我一笑——像高中时一样，我终于把她和那个校服女孩对上了。

她的兴味并没有被这个小插曲搅浑，依然元气满满和我讲着：「跑马拉松最爽的时候，就是超过那些特别壮（zhuâng）的大叔，成就感简直是阶乘式up！」

「你说我男朋友啊，对…是我小学同学，也和咱们一起念的中学，不过他在火箭班，当时也没怎么联系。我六月那会儿不是去北京学雅思嘛，顺便就去了三里屯那个TEDx，结果居然碰见他了；然后年末天津又有一场，我俩就一起约了去，再后来莫名其妙就好了。」

我俩讲话语速一贯偏快，桌上的盘盘碟碟还剩不少，一年半好像已经这么着被囫囵讲完了。结账的时候她拿出鼓鼓囊囊的钱包，带点儿气急败坏地和我解释：「这一钱包的乱七八糟，其实全是拉外援的时候被各个银行要求办的，简直无语。」


道别的时候，我俩站在风里谁都不想先走，吹得彼此鼻头红红。

她伸出手来用力抱我一下：「我很想你。」

我懦弱地出不了声，只好先做逃兵：「到家了给我报个平安。」

「嗯，你也是。」

回家的路上我一直抓着手机，问互联网什么是《蜀素帖》，什么是TEDx，马拉松究竟是多长。寒风吹得人似乎变作半只骨笛，脆弱极了。

她报平安的时候，我正端详着镜子里的自己——大学生活只给了我十六斤肥肉，半脸痘痘，一事无成。虽然我放胆自诩到过很多地方，但匆匆一游，不过是做趟邮差罢了。

同样都是上大学，当时的我还在忧心自己的四级到底能不能低空飘过。

大概这就是传说中，被大学「上」了的吧。


所幸，四级最终有惊无险，我不用再晚人一步了。

我不用抓着「都怪大一不给我考试，高中背的那些全忘光了」的稻草洇渡自己了。


但我无法容忍自己这身赘肉了。体重秤上可怕的数字和少女纤细的小腿交错地摇荡在我眼前，拯救地球的白日梦且先放放，我要先和自己开战。

我不知道这腔三分钟热度，能够我燃烧多久。但编辑部某灵魂画师每天在票圈公然晒扇贝打卡的行为，已经彻底激起了我的斗志——截至我敲下这段话的时候，她放出了连续打卡六十四天的截图。（要知道懒癌如我，早起都坚持不了四天ಠ_ಠ）


看过我们周三推送的人可能会注意到，我们新推出了「打卡群」。基于玩湖的受众，建立微信打卡群，不限形式和目标，互相分享自己的每日打卡。其实「打卡」不过是强制养成习惯的一种手段，对于学英语来说，背单词的重要性我们从小便知；对于健身减肥，坚持运动和健康饮食也是必须的……毕竟，没有什么事情是可以一蹴而就的。大学不同于初高中，再没有老师家长会对你时时刻刻督促鞭策，也没有类似高考的达摩克利斯之剑悬挂在你头顶，「放纵」变得易如反掌。

我们的「定制故事」（后台回复「定制故事」可查看往期精彩）推出后，最吸引人关注的当属梦圆中科院的学霸学长。敬佩之余，我们也希望能激发更多人「动起来」的意愿。无论当初来到湖大的选择是否令你满意，无论你现在是否依然昂扬斗志，我们都希望你可以做到「明天会更好」。

想用打卡督促自己养成良好习惯，却怕自己一人独行道阻且长难以坚持，又怕分享朋友圈会招来纷纷扰扰？既然共享这沙湖之畔，共住这七十万平方校园，我们何不选择聚在一起，分享打卡、互相激励？从新学期开始，从万物复新的春天开始，从一个个小习惯开始，加入「玩湖打卡群」改变自己。



我们可能无法改变身边的环境，但我们可以选择变成崭新的自己。

愿我们终有一天，可以变成脖子长长的水鸟。在朝阳初起的时候，一跃便能衔起最早的天光。
