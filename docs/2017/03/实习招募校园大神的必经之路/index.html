玩转湖大

想买的东西永远只在购物车却狠不下心


舍友和我都是生活简朴的少女

双十一的时候都约定好卸淘宝不剁手

然而

英雄难过美人关

少女难捱双十一
看别人抢到不少优惠券

还是逛了一整天淘宝

购物车已有几十件好物

准备零点大战一番


其实，到了最后，只买了100块左右的东西

“想买的东西，永远都留在了购物车”


自信点，世界也许没那么可怕

大学一哥们喜欢班里学霸女神

但他长相一般

很内向

也没啥过人之处

为了引起女神注意

他向我求助

我建议他去健身

换个发型

拓展兴趣

好好学习

坚持了没几天，就放弃了

后来女神要去台湾交流

多次鼓励他最后一搏

他还是放弃了

比起异地

更怕被拒绝

其实，他早就知道结果，只是不敢面对现实


哥们有追女神的野心，却从不愿付出行动。

人在追求自己恳切的事物时，
往往特别胆怯，越没有自信便越害怕失去。 


悟空：我要这铁棒有何用


认识体院一“不务正业”的哥们

他特爱听《悟空》

总不着调地哼：“我要这铁棒有何用？”

大圣是他从小的英雄

他知道

现在的自己就像大闹天宫的悟空

不得志，很孤独

他渴望一次去闯关的机会


正巧，同一个创业idea

他遇到志同道合的人

巧合太多，知音难得

他们很默契，申请了项目

很快有了自己的外卖店

从线上的营销和店铺设计

充满了悟空的元素

连店名里都有悟空

学生自创的外卖

当然很对学生的胃口

现在，他俩月入过万，得到不少认可


大多时候，缺的是一个契机和平台

去施展才华，放逐野心

 


话不多说，

现在机会就在眼前
