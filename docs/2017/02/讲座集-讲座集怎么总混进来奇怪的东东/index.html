1.快速城镇化下中国城市社会空间重构

主讲人：李志刚

时间 ：03月01日 15点00分

地点：华中师范大学10号教学楼南楼406室

主讲人简介：李志刚，武汉大学规划工学学士，南京大学人文地理学硕士，南安普敦大学城市地理学博士, 中山大学地理科学与规划学院城市与区域规划系教授。在国内外城市地理、城市研究与城市规划领域发表文章100多篇，其中SSCI收录20多篇。参与国内外10多部书籍编写，出版中文专著1部。主持国家自然科学基金项目4项，参与国际和省部级科研项目10多项。


2.如何欣赏昆曲音乐——以《牡丹亭》《长生殿》为例

主讲人：孙建安

时间 ：03月02日晚上19:00

地点：华中科技大学同济校区一号教学楼一大教室

主讲人简介：昆曲作曲家、国家一级演奏员，江苏省演艺集团昆剧院音乐总监。中国民族管弦乐协会竹笛专业委员会常务理事，江苏省音乐家协会竹笛学会副会长。荣获第十三届文华奖“文华音乐创作奖”，首届戏剧文化奖“伴奏奖金奖”、第五届中国昆剧艺术节“优秀作曲奖”等多项荣誉。


（注：此为系列讲座一，系列二会在03月03日晚上19:00 华中科技大学东九楼B202举办。如果有课程冲突或晚归不安全等因素顾虑，周六下午2：30-4：30在武汉市图书馆二楼多功能报告厅也会举办相同内容的讲座。）


3.谁偷走了你的幸福

主讲人：王世朝

时间 ：3月4日上午9:30

地点：湖北省图书馆长江报告厅

主讲人简介：1985年毕业于安徽师范大学中文系，同年志愿支边青海。曾任青海师范大学人文学院古代文学教研室主任、古典文学教授，中国古代文学硕士研究生导师。现为合肥学院中文系教授，汉语言文学专业教研室主任。研究方向：中国古代文学及古代文论。



4.文明之海——从古埃及到拜占庭的地中海文明


展览时间 ：2月24日—5月22日

参观时间：9:00—17:00（周一闭馆，16:00停止入场）

地点：湖北省博物馆 

活动看点：省博展出的部分截取了地中海文明中从古埃及到拜占庭的部分，既有埃及的神秘，也有罗马的享乐，以及腓尼基人在航海上的造诣。展览中的近500件展品都是从意大利20多家博物馆漂洋过海而来的真品。
