方小达就查成绩等问题致校长的一封信

原创： 方小达 玩转湖大

前言



这或许是最后一次我像这样关心母校的情况了，我知道，坚持很难，难在有太多太多外界因素阻碍。我很开心，曾经也发光发热过，也凭借着自己的努力影响过身边的人，影响过母校。现在，所有的校园微信号都标配查成绩和课表以及图书馆藏书，同学们开始关注加权平均分，知道了绩点的存在，如果有同学因为这些能够更加关注学习而不是学校周边的6个商业街，玩转湖大存在的目的就已经达到了；我也曾经让大家关注过图书馆的事情，并且引发了全校范围内的大讨论，这是我第一次或许是最后一次看到全校同学这样的讨论，很开心，玩转湖大为此而感到自豪。


祝大家新年快乐！若有机会，我愿意用我的酒来换得你的关于玩转湖大的故事。

方小达致校长的一封信

尊敬的校长：


您好！


我是湖北大学2010级商学院的一名校友，校园微信公众号“玩转湖大”的创始人，虽毕业三年有余，但仍心系沙湖畔的母校，因为这里有着我的青春，有着我的回忆，有着我亲爱的老师和同学。我是一个恋旧的人，于母校尤甚，每次放假，我都会特地绕道来母校走走；和同学好友相聚，我也会约着在“同学酒楼”和“小重庆”会面。“日思日睿，笃志笃行”这八字校训，就像家训一般督促着我，让我进步。


从小受到的教育告诉我，要以德报怨，我也时时把这四个字记在心中，我总是告诫自己，生活就是一场修行，我所经历的这一切我接受，我不能要求我的母校只给我带来快乐和幸福，而不带来除此之外的东西。我不能够，看到母校的吉祥物是“借鉴”pixiv上画师的作品就认为母校没有为这个形象花心思（毕竟头饰和配色是改了的），看着学校频繁停水停电就觉得后勤集团做得不好（毕竟宿舍楼有空调了），不能够看着图书馆又开始清理考研物品就觉得图书馆处理问题简单粗暴（毕竟图书馆设备都更新了），更不能够，看到仅仅是因为没有办校园网就不得不在评教的时候奔走于图书馆的同学而觉得母校的信息化建设不好（新版的网站做得比我那个时候好看多了）。

“

可是 我真的真的希望

母校能够越来越好

”



这个学期以来，很多学弟学妹在“玩转湖大”的后台向我们打听学校的教务系统网站为什么不能够进去了。为了这件事，我和小编们还特地开了一个会专门研究这个问题寻找原因，小编发现，通过校园网是可以进入教务系统网站的，但是通过校园网之外的网络均不能。起初我以为是学校网站临时维护关闭了外网的访问能力，我以为会贴出通知之类的说明，于是便进入湖北大学官网和母校信息化中心搜寻，但是，没有找到任何相关说明。同样，我委托玩转湖大的小编以及我在母校的老师打听消息，得到的消息是确实在更新系统，但是什么时候能够恢复外网正常访问，不知道，快好了。这些消息打听工作是我学期初的时候进行的，可是直到现在，教务系统网站依旧无法通过外网访问，依旧没有任何相关通知。


幸运的是，湖北大学官方企业微信和湖北大学官方微信截至目前均能够稳定访问教务系统的成绩和课程信息，同时也有其他的微信公众号通过“技术手段”访问教务系统查询相关信息，但是对于除了查成绩和课表之外的需求，依旧无法满足。


我给校长先生您写信，希望您能够帮帮忙催一下教务系统那边的老师，能不能稍微快一点，尽快让教务系统网站恢复外网访问，让那些没有校园网的学弟学妹们不再因此奔波于图书馆机房，不再因此不能够在家里正常使用教务系统。如果是真的快不了，也希望能够出一个通知，告诉学弟学妹们大概什么时候能够好，让他们不再受到这种本没有必要的麻烦。



听说，学校图书馆的设备又更新了；听说，今年的“三星”比往年的更加精彩；听说，学弟学妹又在全国比赛中获奖了。每当通过QQ空间和微信朋友圈看到这些关于母校的消息的时候，远在西子湖畔的我，仿佛回到了水碧琴园，趴在图书馆的座位上，伴随着沙沙的翻书声小憩；仿佛坐在思睿厅的过道上，聆听大师的教诲；又仿佛踮着脚，在学弟学妹们的尖叫和呐喊声中，透着密密的人群观看“十大歌星”总决赛。每每想到这里，我都会打趣地自言自语道：“呵，好处都让现在的学弟学妹们享受了，我们这些老人都被拍到了沙滩上去了，不过，我的经历你们不一定都会经历，以后遇到你们了，就讲你们不知道的，稳的，不虚，哈哈......”


“

如果我和母校的故事

能够一直这样

那该多好啊

”


我睡前和吃饭的时候都有刷手机的习惯，湖北大学官方微信更是被我置顶，生怕遗漏任何有关母校的信息，但是，从这学期开始，我就再也收不到官方微信任何消息了。起初，我以为我的手机坏了，可是微信卸载重装N次后依旧如此，作为微信公众号的运营者的直觉告诉我，我可能被拉黑了，在经过多次仔细反复确认之后，我不得不告诉自己的直觉是正确的。我不知道自己做错了什么受到了母校这样的对待，真的不知道，我很伤心。


作为湖大记者团新媒体部曾经的技术负责人，我谨慎地寻求答案，觉得这里边一定什么“误会”，希望能够知道原因，我委托在记者团工作的玩转湖大的小编，帮忙问一下，小编告诉我，记者团那边很多人都不知道拉黑这件事，同时她告诉我，能够进入官方微信后台操作的人员不是学生。后来我又委托学校的老师帮我了解情况，同样，老师告诉宣传部的老师也不知道有拉黑这件事。我真的很希望有人能够告诉我原因，哪怕把我批评一顿我都心甘情愿，然后让我改正错误，只要别拉黑我让我看不到母校的消息都行，可是到现在都没有人能够告诉我原因。在写这封信的时候，我做了最后一次尝试，几经周折联系上了湖北大学官方微信的负责人之一“差不多大先生”（为保护隐私用化名代替），很可惜，最后他也告诉我不清楚拉黑的事情。


这件事已经发生了一个学期了，在通过其他途径都无法了解原因之后，我只得写信给校长先生您，希望您能帮帮我，让我这个对于湖北大学还有留恋和不舍之情的校友，知道自己为何被母校抛弃。说真的，伤心之余，我更多的是担心，我担心像我这样待遇的校友会不会有第二个、第三个甚至更多。我一个人的受到这种待遇并不会对母校有什么影响，也就是抱怨一两句，但是这样做让那些和我一样关心挂念母校的校友们怎么办，让那些还在母校生活学习的学弟学妹们怎么办，想到这里，我不免心头一痛，如果大家因为这样的事情而对母校有了不好的印象，进而影响母校的声誉，那就真的只得令人嗟叹了。


最后，感谢校长先生能够耐心倾听一个近乎于话痨的校友的文字，我相信您会是一位为母校学生和校友着想的好校长，希望能够早日收到您的回信，如果有机会，我还是会在去杭州前在母校里走走，点上一碗嘉惠园的热干面，回味当年的校园时光。


祝校长先生您，

身体健康，万事如意


祝母校，

鸡鸣琴园红日上

biubiu报晓传佳音


方文达
