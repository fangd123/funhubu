比赛规则

1.建议使用真实姓名参赛报名，以便获奖后核对你的信息并及时联系你，核对证件包括但不限于身份证、学生证等，参与对象必须是全日制高校在校生且非橙光签约作者；
2.作品首发字数不少于5000字，首发时间必须在2017年10月18日0点之后。（发布作品时请务必在“所属活动”中勾选上“校园杯制作大赛”的标签！）；
3.题材不限，作品剧情为作者原创或改编自“橙光正版授权小说库”，作品素材必须全正版。每人可投稿多部作品，但各投稿作品须有显著差异，奖项不重复发放（人气奖除外）；
4.作品字数5000字以上，在评选开始前必须完结至少一条剧情线；
5.活动期间参赛作品不得设置收费项；
6.原创的参赛作品应是未公开发表的作品，即该作品应未在其他任何第三方平台等地发表过；
7.获得奖金的参赛者，橙光将会邀请其与橙光进行签约，奖金会在签约后的次月随分红一起发放。

大赛流程

报名及作品提交阶段

参赛者需登录进行线上报名（须填写真实信息，橙光制作工具的使用方法将会有橙光官方人员在大赛群中进行指导），作品制作完毕后，在发布页面选择【校园杯制作大赛】标签即视为正式参赛。

评审阶段

评委将由橙光公司专业人员组成，以作品创意为主要出发点，对作品进行专业纬度打分。

颁奖阶段

最终决出奖项归属。获奖结果将在官网公布。

评审方式

1.评委将由橙光专业人员组成，以作品创意为主要出发点，参考作品鲜花、点击、收藏等数据进行综合评选，作品没达到奖项水准，奖项将轮空；

2.大赛结束后决出奖项归属，将在官网公布获奖结果；

3.奖项不重复发放（人气奖除外）。
